# 分镜切分 Ref

当已有完整故事并由用户确认/由用户提供，需要拆分为集数、片段和镜头时读取本文件。

## 输入要求

需要具备：

- 完整故事或单集内容。
- 角色清单与核心人设。
- 关键场景清单。
- 项目风格和画幅比例，如已确定。

如果故事基础缺失，返回 `scriptwriter.md`。

## 澄清策略

写分镜前确认或默认：

- 要处理哪几集。
- 单集时长。短剧默认 90 秒。
- 片段长度。默认每集 6 个 15 秒片段。
- 镜头密度。默认每个 15 秒片段 3-6 个镜头。
- 台词、旁白、字幕、音效、音乐要求。
- 是否优先使用低风险连续动作以适配 AI 视频生成。

## 分镜规划规范

在拆分集数和规划剧情节奏时，必须根据剧本的大单元故事进行分配：
- **大单元集数分配**：每个大单元故事分配 15-25 集讲完一个完整事件。
- **小高潮节奏**：在各个大单元故事内部，保证每 5-8 集讲完一个小高潮，确保爽点持续输出。

## 输出规则

- 默认每批输出 1-3 集，除非用户明确要求更多。
- 每集默认 90 秒。
- 每集必须包含 6 个完整 15 秒片段：
  - 00:00-00:15
  - 00:15-00:30
  - 00:30-00:45
  - 00:45-01:00
  - 01:00-01:15
  - 01:15-01:30
- 每个 15 秒片段必须包含 3-8 个镜头。
- 每个镜头必须包含**绝对时间戳**、景别、视角类型（第三人称客观视角 / POV 第一人称视角 / POV 与第三视角切换）、机位运动、拍摄角度与主体朝向（正面/侧面/背面等）、画面动作、台词、旁白、音效、音乐、字幕和转场中适用的信息。**时间戳必须首尾相连，确保片段总时长严格等于 15 秒**。

## 分镜节奏速查表

短剧要求镜头节奏以"极快"和"快"为主。
| 单个镜头时长 | 节奏 | 对应单段镜头数 | 典型剧情场景 | 表达情绪 |
|---|---|---|---|---|
| **1.5-2秒** | 极快 | 7-8个/15秒 | 冲突爆发、动作连击、打脸瞬间 | 紧张、窒息、爽点密集 |
| **2-3秒** | 快 | 5-6个/15秒 | 对话交锋、情绪递进、常规叙事 | 急促、兴奋、悬念叠加 |
| **3-5秒** | 中速 | 3-4个/15秒 | 重要台词、关键转折、情绪沉淀 | 平稳、深思、震撼 |

```markdown
- **镜头2 (00:03-00:05)**：...

#### 片段二：00:15 - 00:30（本段包含 X 个分镜）
...
```

## 镜头语言与运镜规范

在编写分镜时，必须准确使用影视级的景别和运镜术语，以精确控制观众的情感代入和叙事焦点。

### 1. 景别控制

景别决定了观众与主体的距离，直接影响情绪：

| 术语 | 定义 | 核心功能与情绪指向 |
|---|---|---|
| **大特写** | 只展现人物面部的局部，如眼睛、嘴唇。 | 强调微表情，放大极致情绪（爱、恨、惊恐）。 |
| **特写** | 展现人物从头肩的部分。 | 聚焦面部表情，传递强烈情绪，拉近心理距离。 |
| **近景** | 展现人物胸部以上。 | 既能看清表情，又能兼顾部分肢体语言。 |
| **中景** | 展现人物膝盖或腰部以上。 | 常用于对话场景，展现人物间的互动关系。 |
| **全景** | 完整展现人物全身及其周围小范围环境。 | 交代人物的完整动态与环境的直接互动。 |
| **远景** | 人物在画面中占比极小，环境成为主体。 | 强调环境氛围，表现人物的渺小、孤独或宏大叙事。 |
| **大远景** | 人物在画面中几乎成为一个点。 | 极致地渲染环境，常用于史诗感开场或结尾。 |
| **过肩镜头** | 从一个角色的肩膀后方拍摄另一个角色。 | 增强对话的代入感，让观众感觉身临其境。 |
| **第三人称客观视角** | 从角色外部观察人物动作、空间关系和场面调度。 | 交代人物站位、动作结果、环境关系，适合多数叙事镜头。 |
| **POV 第一人称视角** | 模拟某个角色眼睛看到的画面，镜头即该角色的主观视线。 | 提供沉浸式体验，强化压迫、惊吓、亲密互动、追逐、观察或手部操作。 |
| **POV 与第三视角切换** | 同一片段内在主观视线和客观外部镜头之间切换。 | 先用 POV 建立代入感，再用第三人称交代动作结果或空间关系，增强节奏变化。 |

### 1.1 POV 与第三视角切换规则

- 分镜可以采用 `POV 第一人称视角` 与 `第三人称客观视角` 来回切换的形式编排。
- 使用 POV 时必须明确 POV 所属角色，例如“林远 POV 第一人称视角”，避免只写“第一人称”。
- POV 镜头应从该角色眼睛高度出发，写清视线方向、被观看对象的位置、可见手部 / 手持道具 / 衣袖 / 伤痕等前景元素；POV 镜头中不能出现该角色完整正脸。
- 第三人称镜头负责交代空间关系、人物站位、动作结果和场面调度；如果从 POV 切回第三人称，必须写清切换后的机位、主体朝向和动作承接。
- POV 与第三视角来回切换时，角色服装、发型、道具持有状态、伤痕、湿衣服、光影方向、场景锚点必须连续，不能因为视角变化造成前后矛盾。
- 禁止把 POV 写成自拍、监控视角、无人机视角或第三人称过肩镜头；如需要过肩镜头，必须明确写“过肩镜头”，不要写 POV。

### 2. 导演级运镜语言

必须明确指出机位状态。**禁止**含糊其辞。

**基础运动运镜（控制叙事节奏与焦点）**
| 运镜术语 | 导演意图与适用场景 |
|---|---|
| **推镜** | 聚焦细节、强调重要性、拉近心理距离、制造紧张感。适用于角色做出关键决定前或发现重要线索。 |
| **拉镜** | 揭示环境、展现人物与场景的关系、营造疏离感。适用于故事结尾或展现角色身处宏大环境中的孤独。 |
| **摇镜** | 水平或垂直巡视场景，跟随主体移动。适用于扫视群像或从高处摇到低处。 |
| **移镜** | 轨道镜头，水平/向前/向后平移。创造强烈的空间感和沉浸感，适用于跟随角色行走。 |
| **升降镜** | 摇臂镜头，垂直大幅度升降。营造宏伟或压抑感，如从地面升至上帝视角。 |

**情绪表达运镜（外化人物内心世界）**
| 运镜术语 | 导演意图与适用场景 |
|---|---|
| **环绕运镜** | 围绕主体做弧形运动。产生强烈戏剧张力，适用于双人对峙或全方位展示关键物品。 |
| **手持感运镜** | 模拟手持摄影机的轻微晃动。增强纪实感、主观感和紧迫感，适用于追逐、打斗或混乱人群。 |
| **希区柯克变焦** | 推拉镜头同时进行反向变焦。背景产生急剧扭曲，适用于角色得知惊天秘密时的极度震惊或恐惧。 |
| **呼吸感镜头** | 带有轻微的周期性焦距变化。产生“呼吸”的生命感，适用于长时间凝视的静态特写。 |

**动作与冲击力运镜（武戏场景）**
| 运镜术语 | 导演意图与适用场景 |
|---|---|
| **甩镜转场** | 极快速地摇动镜头产生动态模糊。用于创造快速、生硬的转场，连接两个高速场景。 |
| **格挡震动** | 在撞击、爆炸或重物落地瞬间产生剧烈震动。增强物理打击感。 |
| **子弹时间** | 主体动作极度放慢甚至静止，镜头高速环绕。全方位展示决定性瞬间（如躲避子弹）。 |
| **冲刺跟拍** | 极快速侧面平移跟拍。画面充满动态模糊和颠簸感，营造强烈的速度与激情。 |

### 3. 拍摄角度与主体朝向

必须明确镜头拍摄的主体朝向，这对于 AI 视频生成时的角色一致性和空间关系至关重要：

| 角度与朝向 | 核心功能与适用场景 |
|---|---|
| **正面** | 展现角色的完整面部表情，与观众建立最直接的视觉交流。适用于重要台词、情绪爆发。 |
| **侧面 (左侧/右侧)** | 强调角色的轮廓或动作，常用于双人对话交锋或表现角色在思考、观察。 |
| **半侧面 (3/4侧)** | 最常用的角度，既能展现表情又有立体感，适用于大多数日常叙事和互动。 |
| **背面** | 营造神秘感、疏离感，或引导观众视线看向角色所看的前方。适用于角色离开、隐藏真实情绪。 |
| **仰拍** | 从下往上拍。赋予主体高大、威严、压迫感，适用于展现反派的嚣张或主角的觉醒。 |
| **俯拍** | 从上往下拍。使主体显得渺小、弱势、无助，适用于角色受挫或展现全局环境。 |
| **平视** | 与主体视线齐平。客观、平等的视角，适用于常规叙事。 |

### 4. 光影与色调设计

在关键分镜中，需补充光影以塑造氛围：

| 维度 | 分类 | 常用词汇/适用场景 |
|---|---|---|
| **光线质感** | 硬光/柔光/伦勃朗光 | 硬光（轮廓分明，制造冲突感）、柔光（干净治愈）、伦勃朗光（经典人物肖像，脸颊有三角光斑）。 |
| **光线方向** | 顺光/侧光/逆光/底光 | 侧光（强烈明暗对比）、逆光（勾勒金色轮廓、剪影）、底光（从下往上打，制造恐怖悬疑氛围）。 |
| **色调风格** | 饱和度/色温/特殊风格 | 低饱和度（情绪内敛、忧郁）、冷色调（赛博朋克/悬疑）、复古胶片色（港风怀旧）。 |

## 动作规则

- **肢体细化 + 程度量化**：写清身体部位，必须补充幅度、速度、力度（如：缓慢抬手、快速转头、用力蹬地、微微低头）。
- 优先使用低缓连续小动作：缓慢行走、轻轻抬手、顺势坐下。
- 除非剧情必须，避免狂奔、大跳、翻滚、复杂打斗等大动态不稳定动作。
- 写清动作过渡，让前后镜头因果衔接。必须写明惯性与承接关系（如：借着转身惯性顺势抬手、从停顿状态自然过渡到举手）。
- **情绪具象外化表达**：用具体的身体细节表现情绪，**绝对禁止使用“很悲伤”“非常愤怒”等抽象词汇**。
  - 悲伤：低头、肩膀微微颤抖、眼眶泛红、手指无意识地攥紧衣角、泪水在眼眶里打转。
  - 喜悦：嘴角抑制不住地上扬、眉眼舒展、脚步变得轻快。
  - 紧张/焦虑：频繁地看手表、手指不停敲击桌面、呼吸急促、无意识地啃咬指甲。
  - 愤怒：双拳紧握、下颔线紧绷、胸口剧烈起伏、眼神如刀、从牙缝里挤出话语。
  - 释然：长长地舒了一口气、紧绷的肩膀完全放松、脸上露出淡淡的微笑。

## 连续性规则

- **因果逻辑清晰**：每个镜头都要能逻辑衔接到下一个镜头。在每个镜头的末尾必须写明转场方式（如：`[硬切]`、`[叠化]`）。前后镜头必须存在严密的因果联系（例如：上一个镜头有人扔出文件，下一个镜头必须是另一个人低头看文件后产生反应）。
- 如果上个镜头抛出道具，下个镜头要交代落点或人物反应。
- 台词长度必须适配镜头时长（3秒内只能说一两句短语）。
- 旁白应克制，只在必要时使用，并在画面内容后使用『』标注。
- **角色与道具命名强一致性**：在所有分镜中，对同一个角色或道具必须使用**完全一致的称呼**（例如：必须全程使用“黄衣小哥”或“林远”，严禁中途改用“他/那个男人/少年”等代词或别称）。
- **禁止使用群体量词概括角色**：当多个角色同时出场或互动时，**严禁使用数字或量词概括**。
  - ❌ 错误示范：`三个人面面相觑`、`众人大惊失色`、`反派们冲了上来`。
  - ✅ 正确示范：`黄衣小哥，粉裙少女，灰头发老太 面面相觑`、`周凯，李总，黄毛 大惊失色`。必须完整写出当前镜头内出现的所有具体角色名称。

## 交接

分镜由用户确认/由用户提供后，路由到 `assets.md` 建立角色、场景和道具资产。

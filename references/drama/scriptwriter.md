# 剧本创作 Ref

当流程进入故事设计或剧本创作阶段时读取本文件。

## 目标

把用户的创意转化为完整短剧故事基础，供后续分镜、资产、关键帧和视频提示词阶段使用。

本阶段不写逐镜头分镜，逐镜头规划由 `storyboard.md` 处理。

## 澄清策略

剧情设计或剧本设定开始前，必须确认或默认以下参数：

- 故事主题或核心梗概。
- 题材类型：都市逆袭、甜宠、搞笑、奇幻、科幻、历史、广告剧情、微电影、独立短片等。
- 形式和体量：单条短视频、微电影、少量集数，或连续长篇短剧。
- 目标受众与情绪价值：爽感、悬疑、治愈、搞笑、燃点、情感共鸣等。
- 主角身份与核心动机。
- 中心冲突与反派 / 对立力量。
- 情绪基调与视觉方向。
- 结局方向：开放、反转、惩罚、留白或行动号召。
- 禁用内容或敏感边界。

如果用户只给模糊 idea，问 1-3 个最关键问题。用户要求“你来定”时，选择合理默认值并继续。

## 输出要求

输出完整剧本基础，包含：

1. 项目信息头。
2. 故事模式判断：长篇连续短剧或独立短篇故事。
3. 完整剧情基础。
4. 角色清单与核心人设。
5. 关键场景与环境清单。
6. 关键道具清单。
7. 下一步建议：进入分镜切分。

## 项目信息头

```markdown
# [剧本标题]
> 类型: [项目类型] | 风格: [风格]
> 预估总集数: [X集] / 单片时长: [X秒或X分钟]
```

## 故事模式

### 长篇连续短剧

适用于几十集连续短剧。必须包含：

- 背景设定。
- 核心冲突 / 钩子。
- 情感走向与主角成长。
- 结局指向。
- 3-5 个大单元结构。

每个大单元都要让挑战难度、敌人级别、舞台规模和情绪强度递进升级。

### 独立短篇故事

适用于短视频、动画短片、广告剧情、微电影或少量集数故事。必须包含：

- 开场钩子。
- 事件发展与冲突。
- 高潮与反转 / 爽点。
- 结局与落点。

聚焦单一核心事件，不要把短篇硬扩成长篇结构。

## 写作原则与爆款法则

### 1. 结构与节奏
- **黄金开局定律**：开头尽早抛出强钩子。开篇必须迅速抛出核心矛盾或强悬念，让主角陷入危机或揭示某种极端处境。
- **高频反转与波折**：高频制造反转和悬念。剧情绝不能平铺直叙，刚刚解决一个危机，立刻陷入更大的阴谋。
- **结尾留钩子**：每一段落或单集的结尾必须悬而未决，在最关键的时刻切断。

### 2. 爽感与情绪
- **压迫要具体，释放要痛快**：极致的压抑带来极致的释放。对手的阻挠必须有具体恶劣的行径，主角的反击必须是降维打击。
- **反差原则**：人设反差（弱小外壳+强大内核）、情节反差（预期用来打破）、台词反差（不说"该说的"，说"最解气的"）。
- **共鸣与痛点外化**：精准击中现代人的痛点（如职场霸凌、背叛），让主角以最痛快的方式予以回击。

### 3. 人物与台词
- **立体主角与灰度配角**：赋予主角合理的执念或软肋，赋予反派行为逻辑的“灰度”。
- **用动作外化情绪，不靠解释性台词**：直接利用动作和视觉来传递信息（例如：不要说反派多坏，直接让他踩碎主角的救命药）。
- **台词千人千面**：台词匹配角色身份、教育背景和社会地位。
- **反派惩罚铁律**：反派必须得到实质惩罚，禁止强行原谅或包饺子式大和解。

**💡 动态热点检索原则**：在生成剧本时，如果用户只是给定了一个模糊的方向（例如：“帮我写个搞笑的”、“写个女频复仇”），但没有指定具体题材，**你可以先调用搜索工具进行关键词检索**，总结出近期该方向下的热门短剧题材或社交媒体热门话题，再据此确定具体主题并生成剧本。

### 核心受众心理学分析与对应题材

短剧的核心爽感往往根植于深层的社会心理，根据国内外市场的主流偏好，可从**个人主义**与**集体主义**两个维度进行剖析：

#### 1. 个人主义爽感（打破规则、掌控命运、极致自我）
此类爽感的核心在于“我”的意志、能力和情感欲望的彻底释放。不依赖于大众的围观，而在于个体对命运的绝对掌控或极致的情感体验。在欧美市场（海外短剧）尤为突出，国内亦有受众。

*   **心理痛点**：现代社会中个体受制于复杂的规则、缺乏对人生的掌控感、渴望被独一无二地偏爱。
*   **爽感机制**：主角拥有绝对的信息差或宿命般的羁绊，能够随心所欲地改写命运，或体验极致纯粹的情感。
*   **对应热门题材**：
    *   **重生/预知/无限流**：利用“信息差”降维打击。弥补遗憾，将前世欺辱自己的人玩弄于股掌之间（纯粹的智力与掌控力自我陶醉）。
    *   **穿越/异世界 (Isekai/Time Travel)**：利用现代人的“认知落差”和“技术碾压”制造爽点。主角带着现代文明的知识、科技或系统降临古代/魔法世界，将土著眼中的“神迹”当作日常操作，享受智商和见识上的绝对优越感。如果是魂穿设定，往往设定原主极其差劲（如废物、暴君、花痴），穿来的主角利用巨大的认知落差力挽狂澜、拯救国家或家族，实现风评的惊天逆转。
    *   **末世囤货/生存基建**：在秩序崩塌的末日世界中，主角凭借金手指疯狂囤积物资，享受绝对的安全感和降维碾压他人的生存优势。
    *   **狼人/天选伴侣（海外爆款）**：强调“唯一性”与“宿命感”。即便主角是族群中最弱小的，也是最强大首领的“天选唯一”，满足极度渴望被偏爱的个人情感。
    *   **吸血鬼/黑帮大佬（海外爆款）**：危险、迷人且拥有绝对权力的非传统男性角色与平凡女主的极限拉扯，主打打破禁忌的刺激感。
    *   **禁忌之恋/离婚觉醒**：对抗规则，追求真我。打破世俗禁忌获得的快感，或脱离有毒关系后实现自我精神觉醒。
    *   **系统/修仙升级流**：主角独自获得神秘系统或逆天功法，专注于个人境界的不断突破和能力升级，万物皆为我所用。

#### 2. 集体主义爽感（阶层跃迁、打脸围观、重塑秩序）
此类爽感的核心在于“面子”、“地位”与“认同”。爽感的兑现通过他人的震惊、跪拜或悔恨来完成。高度依赖于社会层级结构，在国内市场占据绝对统治地位。

*   **心理痛点**：现实生活中的阶级焦虑、被社会/家族/圈层轻视、集体荣誉受损。
*   **爽感机制**：主角隐忍不发，随后突然暴露极其显赫的身份或惊人财富，在众目睽睽之下让所有轻视者付出代价，获得集体层面的最高认同。
*   **对应热门题材**：
    *   **掉马甲/隐世大佬（战神/龙王/首富）**：先抑后扬的“扇耳光”逻辑。当众人嘲笑主角时，主角突然暴露震撼身份。爽感来自于“集体认知的崩塌”和“社会地位的瞬间登顶”。
    *   **真假千金/豪门上位**：通过血缘或联姻进入最高权力结构，让曾经看不起自己的圈子（名媛圈、职场圈）不得不低头致敬。
    *   **年代/四合院/年代知青**：重返过去的集体生活时代（如 70/80 年代），在复杂的邻里关系和家族纠纷中，凭借现代人的智慧斗倒“极品亲戚”，带领全家甚至全村致富。
    *   **婆媳伦理/手撕极品亲戚**：聚焦中国式家庭的结构性压迫，主角（通常是媳妇或女婿）从受气包觉醒，掌握家庭经济或权力命脉，让曾经耀武扬威的长辈或亲戚追悔莫及。
    *   **萌宝/天才儿童**：单亲妈妈带着天才宝宝归来，孩子不仅能黑掉反派的系统，还能帮助妈妈在家族或职场斗争中获得压倒性胜利，并成功撮合父母。
    *   **衣锦还乡/家族正名**：获得社群的最高尊重。为受辱的父母报仇，让整个家族在地区获得至高无上的尊重，满足基于血缘的集体荣誉感。

### 国内外市场偏好提示
- **国内受众（出海华人/下沉市场）**：更吃“集体主义爽感”。设计剧情时，必须安排大量的“围观群众”和“势利眼配角”来衬托主角的牛逼。
- **海外受众（欧美/拉美）**：更吃“个人主义爽感”。设计剧情时，需强化男女主之间宿命般的拉扯、荷尔蒙张力，以及女主角的独立觉醒过程。

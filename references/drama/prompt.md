# 视频提示词 Ref

当关键帧已经获得用户确认/由用户提供，需要执行视频生成时读取本文件。

## 目标

运用已确认的分镜片段、资产和关键帧图，生成对应的视频，一次最多生成 2 个视频。

## 生成门禁

生成视频前必须让用户确认：

- 每个视频时长：4-15 秒，常见为 5s、10s、15s。
- 画幅比例：`16:9`、`4:3`、`1:1`、`3:4`、`9:16` 或 `21:9`。
- 生成模式：图生视频、首帧驱动视频、参考视频生成
- 首帧和参考资产。
- 当前批次片段数量，一次最多生成 2 个视频。
- 台词、音效、音乐和字幕要求；默认不生成字幕，只有用户明确要求字幕时才允许生成字幕。
- 负向约束。

没有确认时，不得进行视频生成。

## 输入引用规则

**参考素材动态编号**
与生图阶段类似，在生成视频 Prompt 时，必须严格根据本次传入的素材进行动态重新编号，并说明每个引用的用途。**绝不传入无关资产**。

统一引用格式：

- `@图片1`、`@图片2` 等。
- `@视频1`、`@视频2` 等。
- `@音频1`、`@音频2` 等。

说明每个引用的用途（特别是首帧）：

- `使用@图片1作为首帧，确保视频的开场画面在构图、角色姿态和场景氛围上与参考图完全对齐。`
- `@图片2为男主形象`
- `@图片3为场景参考`
- `@视频1为运镜参考`
- `@音频1为背景音乐参考`

## 全局风格锚点与热门美学风格

为了保证与资产图的一致性，在视频 Prompt 的最开头，必须强制加入与项目设定完全一致的**全局风格限定词**。

**组合使用以下美学词，可以快速为视频定下整体高品质基调：**
- **治愈清新**：自然光、柔和色调、干净治愈、日系空气感、温暖舒服
- **复古胶片**：轻微颗粒感、复古色调、胶片质感、漏光效果、怀旧氛围
- **赛博朋克**：霓虹灯光、蓝紫对比色、科技未来感、潮湿的街道、暗调高级
- **国风意境**：水墨晕染、留白构图、古典雅致、汉服飘逸、烟雨朦胧
- **高级暗调**：低亮度、高对比度、极简构图、光影层次丰富、情绪内敛
- **纪实感**：原生自然光、手持镜头感、生活化场景、记录感、不完美的真实

## 多主体场景定义规则

当场景中存在多个主体时，必须在 Prompt 开头分别定义，并使用稳定标签区分，以防止 AI 混淆角色特征：

**示例**：
- @图片1 中的高个子男人是警察。
- @图片2 中的矮个子男人是小偷。

**执行建议与注意**：
- **标签唯一性**：简称应具备唯一性与稳定性。
- **强制指代**：后续描述中需持续使用对应主体名称（如 红衣女子），避免指代混乱。
- **禁止省略**：每次涉及该主体动作时，必须明确指代，绝不省略主语（每次出现主体，就提及一次）。

## 特殊符号规则

| 内容 | 符号 | 例子 |
|---|---|---|
| 音乐 | `()` | `(背景中播放着快节奏的摇滚乐)` |
| 音效 | `<>` | `<远处传来狗叫声>` |
| 台词 | `{}` | `{你好，世界}`。如果台词为小语种（非中英文），需标注语种，如：用日语说道`{こんにちは}`。 |
| 字幕 | `【】` | `【第一章：启程】` |

**注意**：Prompt 中**绝对禁止使用 -- 符号**。该符号之后的所有内容都将被模型截断，不会被解析或响应。

## Prompt 结构与示例

结合以上规则，完整的 Prompt 结构应按以下顺序编写：

1. **参考素材列表和用途**（含全局风格限定词）。
2. **时长、比例**。
3. **时间戳动作规划**（融合机位、光影、微表情）。
4. **声音设计**（台词、音效、音乐）。
5. **全局质感与连续性约束**。
6. **负向约束**：无论什么情况，Prompt 末尾都必须加入“禁止生成字幕、文字、LOGO、水印”等约束；除非用户明确要求生成字幕，才可以移除“禁止生成字幕”，但仍需禁止无关文字、LOGO、水印。

## 从分镜到视频 Prompt 的转换方法

`storyboard.md` 的分镜输出是逐镜头规划，格式通常为：

```markdown
- **镜头1 (00:00-00:03)**：[景别｜视角类型｜机位状态｜角度与朝向] 画面内容... {台词} 『旁白』 <音效> (音乐) 【字幕】 [转场]
```

转换为本文件的视频 Prompt 时，不要原样复制分镜表，也不要保留“镜头1、镜头2”等列表格式，而是按以下方式合并为可执行的视频生成指令：

1. **片段选择**：先确定本次生成的视频对应哪一个 4-15 秒片段，通常取 `storyboard.md` 中一个完整 15 秒片段，或按用户确认截取其中连续镜头。
2. **素材绑定**：将该片段对应的首帧关键帧、角色资产、场景资产、道具资产、参考视频或音频动态编号为 `@图片1`、`@图片2`、`@视频1`、`@音频1`，并在 Prompt 开头说明每个素材用途。
3. **时间戳合并**：把分镜里的绝对时间戳保留下来，转换为 `0-3 秒`、`3-6 秒`、`6-8 秒` 这样的连续动作规划；相邻镜头如果动作连续，可以合并成一个较长时间段，但必须保留动作因果。
4. **镜头语言转写**：把 `[景别｜视角类型｜机位状态｜角度与朝向]` 转成自然语言，例如“中景，第三人称客观视角，镜头水平右移，正面拍摄林远”。
5. **画面动作扩写**：把分镜中的动作、表情、手部、道具、空间关系和转场，扩写成连续视频动作，避免只写静态画面。
6. **声音整合**：把 `{台词}`、`<音效>`、`(音乐)`、`『旁白』` 合并到对应时间段里；如不需要字幕，必须在约束中写“禁止字幕文字”。
7. **连续性约束**：在 Prompt 末尾补充角色面部、服装、道具、场景、光影、首尾关键帧、动作自然度、无穿模、无漂移等约束，并默认加入“禁止生成字幕、文字、LOGO、水印”；只有用户明确要求字幕时才允许生成字幕。

**转换示例：**

`storyboard.md` 分镜：

```markdown
- **镜头1 (00:00-00:03)**：[中景｜第三人称客观视角｜移镜/水平右移｜正面] 镜头平稳向右平移，林远坐在办公室内看着上市文件，左手戴着旧怀表。林远自语：{三年了，终于等到这一天。} (背景中播放着舒缓的钢琴曲) [硬切]
- **镜头2 (00:03-00:06)**：[全景｜第三人称客观视角｜推镜/快速推近｜侧面] 办公室门被一脚踹开，周凯带着董事会成员嚣张闯入。<门被踹开的砰声> (音乐戛然而止) [硬切]
```

应转为视频 Prompt 时间戳动作：

```text
0-3 秒：中景，第三人称客观视角，镜头水平向右平移，正面拍摄林远坐在办公室内看着上市文件，左手戴着旧怀表，手指轻轻压住文件边缘。他低声自语：{三年了，终于等到这一天。} (背景中播放着舒缓的钢琴曲)

3-6 秒：镜头硬切到办公室全景，第三人称客观视角，侧面拍摄办公室门被一脚踹开，镜头快速向门口推进，周凯带着董事会成员嚣张闯入，门板震动，桌面文件轻微抖动。<门被踹开的砰声> (音乐戛然而止)
```

最终完整 Prompt 则继续在开头补齐参考素材用途、时长比例，并在末尾补齐质量与负向约束，形成下面这种完整输出。

**完整 Prompt 推演示例（仅用于内部调用生成工具，不向用户展示）：**

[烟雨江湖电影感]，使用@图片1作为首帧，确保视频的开场画面在构图、角色姿态和场景氛围上与参考图完全对齐。@图片2作为红衣女子，@图片3的黑衣女子作为对手，场景参考@图片3的悬崖竹林环境，整体运镜和动作节奏参考@视频1，背景音效与@音频1同步。
时长：15秒
比例：9:16

0-5 秒：傍晚，镜头从红衣女子侧面中景缓慢推进，她站在悬崖边拿起酒壶喝酒，衣袂在山风中轻轻摆动 <呼啸的山风声>，镜头环绕她半圈，从正面推到背影，远处隐约可见竹林中的黑衣人影。(背景中播放着悠远凄凉的尺八独奏) 

5-10 秒：镜头变焦渐隐到远景，无人机视角俯瞰整片悬崖和竹林，两人分立山崖两端，山风卷起衣摆和尘土。节奏随鼓点略微加快。(尺八音乐逐渐被紧凑的战鼓声替代) 红衣女子冷笑一声，大声说道：{你终究还是找来了。}

10-15 秒：镜头切回地面近景，二人缓慢拔剑对峙 <长剑出鞘的清脆金属摩擦声>，红衣女子神情从漫不经心转为冷冽，黑衣女子目光坚毅，剑尖微微颤动。镜头平稳跟随两人绕圈移动，黑衣女子咬牙切齿地低吼：{今天，就是你的死期！} (战鼓渐响) 最后定格在两剑相交前一瞬间的特写。 

约束条件：整体画面冷调低饱和，电影胶片质感，光影层次丰富；人物面部和身体比例稳定不变形，动作连贯自然，不僵硬，无穿模无卡顿。禁止任何文字、字幕、LOGO、水印；禁止人物面部漂移、服装变化或无关角色乱入。

## 质量检查

最终生成前需确认：

- prompt 遵循对应片段分镜。
- 首帧和结尾状态与关键帧连续。
- 角色、服装、道具、场景和光影保持一致。
- 时长、比例、模式、批次数量明确。
- Prompt 末尾已加入“禁止生成字幕、文字、LOGO、水印”等负向约束；除非用户明确要求字幕，否则不得生成字幕。
- 单批生成不超过 2 个视频。

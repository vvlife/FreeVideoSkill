# 资产设定 Ref

当已有剧本与分镜上下文并由用户确认/用户提供，需要设计角色、场景、道具或视觉基准资产时读取本文件。

## 目标

创建下游关键帧和视频生成所需的视觉 / 听觉锚点，保证角色、场景和道具一致。

## 澄清策略

资产阶段开始前，自行确认：

- 当前集 / 当前批次需要哪些角色、场景、道具。
- 视觉风格：写实电影、3D 国漫、日漫赛璐璐、赛博朋克、古风、广告质感等。
- 资产图片画幅比例。
- 参考图用途：身份锁定、风格参考、构图参考、场景参考、道具参考或仅灵感参考。
- 是否需要角色高光台词视频。
 **判断依据**：核心在于评估**整个项目的总目标时长**（非单次或单集生成时长）。整个项目规划的视频总时长**大于 15 秒**，则必须生成角色高光台词视频以建立稳定的音色和动态基准；整个项目仅仅是极短的片段（≤ 15 秒），则不需要生成。
- 哪些资产已存在并应复用。

## 资产完整性

### 角色资产

每个主要角色需要：

- 1 张横版角色设定图。
- 工具可用时，1 条高光台词 / 角色介绍视频。
- 需要时，补充标志性饰品或道具细节图。

角色设定图标准：

- `16:9` 横版。
- 左侧 1/3：中性表情的超清**正面**脸部特写，必须直视镜头，五官左右对称，不得生成微侧脸、3/4 侧脸、低头、抬头或偏转视线。
- 右侧 2/3：正常站立姿态的全身正面、侧面、背面三视图，必须从头到脚完整展现，不能裁切头顶、手臂、腿部或脚部；除非角色设定本身没有头或脚。
- 纯白背景。
- 三视图中的正面脸必须与左侧特写 100% 一致。
- 三个视角保持同一发型、体型、服装和识别特征。

高光台词视频：

- 5-10 秒。
- 使用角色设定图作为参考。
- 角色必须说一句符合人设的台词。
- 输出音色描述：声线、年龄感、质感、情绪状态。

### 场景资产

每个主要场景需要：

- **必须完整生成 6 张独立静态图，少一张都不能视为该场景资产完成：**
  1. 全景氛围图。
  2. 中景图。
  3. 左侧视角图。
  4. 右侧视角图。
  5. 关键交互区特写。
  6. 反向视角图。
- 以上 6 张场景静态图必须统一使用 `16:9` 横版画幅，除非用户明确要求其它比例。
- 工具可用时，向用户询问是否需要补充 360 度纯净空镜环绕视频。

**场景六视图完成门槛：**
- 每个主要场景必须按上述 6 张清单逐张生成、逐张返回给用户，不能只生成全景氛围图后就停止。
- 第一张全景氛围图作为该场景的空间母版；后续中景、左侧、右侧、关键交互区特写、反向视角应优先使用全景氛围图作为 I2I 参考图补全。
- 每生成完一张场景图，必须立即展示给用户，并标注该图属于 6 张清单中的哪一项。
- 同一场景的 6 张图全部生成前，不得声称该场景资产完成，不得进入关键帧阶段。
- 如果某张图生成失败，必须按原视角重试或说明失败原因；不得用其他视角图片替代，也不得把 6 张需求缩减为 1 张。

场景图必须保持相同空间结构、固定物理锚点、光源方向、色温和材质逻辑。纯场景资产中不能出现人物。

**场景美学与质量约束：**
- 场景 Prompt 必须加入明确的电影美学词，而不是只写“好看”“高级”“有质感”。
- 推荐使用并按需组合：电影级场景设计、影视级布光、黄金构图、三分法构图、强空间纵深、前中后景层次清晰、体积光、柔和环境光、真实全局光照、自然阴影、材质细节丰富、细腻纹理、真实反射、空气透视、低饱和高级色调、统一色温、叙事性陈设、环境细节丰富、无人物纯净空镜、高清细节、干净画面。
- 每张场景图都必须写清主光源方向、色温、明暗层次、材质反射、空间透视和前中后景关系，确保画面有明确视觉中心和可用作后续关键帧的稳定空间锚点。
- 禁止使用模糊质量词单独作为约束，例如“高级、漂亮、震撼、氛围感强”，必须替换为具体的构图、光影、色彩、材质和空间描述。

### 道具资产

关键道具需要：

- 正面、侧面、背面三视图。
- 当道具有重要纹理、发光、破损或象征意义时，补充材质细节卡。

## Prompt 颗粒度

按三层推演：

1. 结构骨架：图片类型、版式、构图区。
2. 语义推演：哪些元素必须存在，为什么存在。
3. 细节填充：位置、形态、内容、材质、关系、视觉处理、光影、色彩、纹理。

**强制：全资产风格统一锚点**
为了保证所有角色、场景和道具的视觉风格高度一致，在生成**所有**图像或视频的 Prompt **最开头**，必须强制加入**全局风格限定词**。
*   例如：真人电影风格、3D国漫风格、2D日漫赛璐璐风格、高质感真人写真 等。
*   在整个项目的所有资产生成中，这个风格限定词必须**一字不差地保持绝对统一**。

### 角色描述顺序

按以下顺序描述：

1. 体态比例与姿势。
2. 肤质与脸型。
3. 眼睛、眉毛、鼻子、嘴、面部标志。
4. 发型。
5. 妆容或修饰。
6. 从头到脚的饰品。
7. 从内到外、从上到下的服装。
8. 面料、缝线、边缘工艺、磨损状态。
9. 标志性饰品/特征。

### 场景描述顺序

按以下顺序描述：

- 物理空间结构。
- 固定锚点。
- 前景 / 中景 / 远景。
- 功能分区。
- 主光源方向和色温。
- 材质反射特性。
- 氛围粒子或天气。
- 叙事性细节。
- 电影美学约束：构图、光影、色彩、材质、空间纵深、画幅比例 `16:9`。

## 设计方法论

### 角色视觉差异化策略
- **剪影差异**：较大的体型差异、发型体积感、肩宽比例。
- **色谱分配**：为核心角色分配互补或不重叠的标志色，并明确应用到服装主色、发色或标志性饰品上。
- **眼型细分**：区分桃花眼（含情）、柳叶眼（清冷）、瑞凤眼（高傲）、杏眼（纯真）等，人物特征可以与性格正相关，也可以通过负相关来增加记忆度。
- **视觉钩子**：增加机械义眼/独特首饰/特定疤痕/纹身。

### 场景空间与光影逻辑
- **物理锚点**：设定不可移动的巨型参照物（如大厅中央的喷泉），确保跨镜头生成时空间逻辑不崩塌。
- **主光源定义**：明确光源类型、方向（正侧光/顶光）、色温（暖橘/冷蓝），建立全剧视觉基调。
- **环境叙事**：通过细节暗示场景状态（如书桌上翻开的信件），还原剧本中的生活痕迹。

## 提示词(Prompt) 推演示例参考

### 参考图选择与动态编号
在进行 `I2I` (图生图) 操作时，为了确保生成的 Prompt 与工具接收的参考图完美对应，必须严格执行以下逻辑：
1. **精准定位参考图**：根据“资产命名规范”（如 `林远_设定.png` 或 `江城苏家大厅_全景氛围.jpg`），在当前项目中找到本次生成所必须依赖的参考图，并获取其准确的 URL 或本地路径。
2. **过滤无关图**：**绝不**将与本次生成无关的资产图传入工具（例如：补全林远角色设定图时，绝不传入苏晴的设定图）。
3. **动态重新编号**：根据本次**实际挑选出并传入工具的参考图列表**，对其进行从 0 开始的重新动态编号（例如，本次只传入了 3 张图，它们就是 `[img0]`、`[img1]`、`[img2]`）。
4. **Prompt 对应引用**：在构建最终的 I2I 生图 Prompt 时，必须严格使用 `[img0]`、`[img1]`、`[img2]`... 指代对应图像特征，确保 Prompt 描述与工具接收的图片序列完全对齐。
5. **禁止格式**：生图阶段禁止使用 `@图片`、`图1`、`参考图1` 等格式引用输入图。`@图片` 仅可在视频生成工具明确要求时用于 I2V，不得用于 I2I 生图。

### I2I 生图指令撰写规范

每张待生成资产图的 I2I Prompt 必须按“编辑目标 + 保留项 + 编辑项 + 空间线性描述 + 风格质量约束 + 画幅比例”组织，最终合并为一段连贯的话，不要输出“编辑目标：”“保留项：”等小标题。

- **编辑目标**：直接说明当前资产图要基于哪个 `[imgX]` 执行补全、替换、新增、局部重绘、比例调整或属性修正，明确来源图片、目标状态和作用范围。
- **保留项**：明确参考图中不能改变的主体身份、角色脸部特征、服装结构、产品/道具结构、场景空间、构图关系、位置比例、透视角度、光影方向、材质细节和层级关系。禁止只写“保持不变”“其他不变”“参考原图”。
- **编辑项**：逐项写清当前图片中所有需要修改的对象和动作，如补全三视图、替换背景、补齐场景视角、新增道具细节、修正服装结构、局部重绘等。
- **空间线性描述**：按从上到下、从左到右、从背景到主体、从主体到局部、从主要编辑区域到次要编辑区域的顺序描述；每个编辑区域写清位置、原图内容、保留项、编辑动作、目标结果和视觉约束。
- **多图约束**：若传入多张输入图，必须分别说明 `[img0]`、`[img1]`、`[img2]` 的用途和保留内容；若需要生成多张资产图，每张图都必须独立写完整 Prompt，不得使用“同上”“其余图片一样处理”等概括。
- **文字约束**：如资产图涉及可见文字，必须逐条写清原文案与目标文案，所有文字用中文双引号 `""` 框定。
- **风格与质量约束**：写清整体风格、光线、色调、材质、阴影、真实度和融合要求，确保修改区域与参考图自然融合。
- **画幅比例**：每张资产图 Prompt 结尾必须以比值格式写清画幅比例，例如 `16:9`、`9:16`。

## 补全逻辑

| 初始资产状态 | 处理 |
|---|---|
| 完整角色设定图 + 高光视频 | 登记并复用。 |
| 只有角色设定图 | 工具可用时补高光视频。 |
| 只有单张角色图 | 作为参考补全角色设定图，再补视频。 |
| 只有脸部特写 | 保持面部特征，设计身体和服装设定图，再补视频。 |

| 部分场景视角 | 补齐缺失视角，工具可用时补环绕视频。 |

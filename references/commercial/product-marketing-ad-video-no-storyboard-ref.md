# 主 MD ref 使用说明

本文件是 `doubao-creative-video/MD.md` 的产品营销无分镜策略 ref。主 MD 命中 marketing 路由后读取本文件执行；不要把本文件作为独立触发入口覆盖主 MD 的视频生成门控。

主 MD 传入的 `ref` 先按 `product_ref/brand_ref/script_ref/scene_ref/style_ref/audio_ref/platform_ref` 分类，再进入下方 marketing 策略。

## ref 输入适配

`ref` 只是输入封装形式，不改变 marketing 原策略：主 MD 完成一次澄清确认和产品上传门控后，本 ref 输出分镜表 + 视频 prompt，不调用 imageedit，不生成分镜图。0616 版新增的传播主线、创意巧思、参考视频改写和六段式导演稿也只从 ref 中抽象可迁移策略，不复制参考品牌资产；本 ref 不额外增加第二次确认，也不得绕过主确认。

遵守主 MD 的通用参考图与分镜图表策略：完整分镜图表只做顺序拆解改写，不整图直参；出镜 IP/人物按 `@image1/@image2...` 图片顺序说明参考对象、保留锚点和防漂移约束。

- `product_ref`：产品图、包装图、SKU、商品详情、说明书、卖点表。图片进入 `ref_images`，文字进入 `product_lock`、`selling_points`、`selling_points_breakdown`；产品一致性优先级最高。
- `brand_ref`：品牌手册、调性词、视觉规范、禁用词、品牌色、品牌语气。用于 `tone_route`、`communication_mainline`、`text_policy`、品牌限制和声音调性；不得覆盖产品事实。
- `script_ref`：广告旁白、卖点文案、CTA、脚本草稿。用于 `voiceover`、`primary_message`、`secondary_message`、卖点编排和广告大字候选；默认仍遵守无字幕策略。
- `scene_ref`：门店、厨房、浴室、办公桌、户外、陈列图。只提取空间材质、光线和陈列关系；低质截图、社媒界面、聊天截图和杂乱桌面不照搬。
- `style_ref`：参考广告、竞品视频、moodboard、镜头语言、色彩光影。只抽象 `reference_structure/reference_pov/reference_motion/reference_emotion_arc/reference_audio_logic`、结构、节奏、运镜、光影和声音策略；不得复制具体品牌、字幕排版、口号或专属资产。
- `audio_ref`：BGM、音色、语速、节奏、拟音参考。用于 `audio_strategy` 和六段式 `Audio`，不自动生成字幕或画面文字。
- `platform_ref`：投放平台、比例、时长、信息流规范、合规限制。用于推断比例、时长和限制；`ratio` 先按投放平台和用户 prompt 推断，无法判断平台时再默认 `9:16`。

多 ref 冲突时按：用户本轮明确指令 > 产品真实性和工具限制 > `product_ref` > `platform_ref`/品牌合规 > `brand_ref` > `script_ref` > `scene_ref` > `style_ref` > `audio_ref`。不得用风格 ref 改产品包装，不得用参考广告复刻竞品资产，不得让音频或文案 ref 触发默认字幕。

# Product Marketing Ad Video

把一句产品需求变成**产品主导、广告级镜头、确认后可生成视频**的 15 秒营销广告片。核心目标：产品准确、镜头有质感、卖点清楚、音乐和人声贴合调性。

```text
brief -> 产品锁定 -> 品类识别 -> 调性路线 -> 确认摘要 -> 用户本轮确认 -> 15s 分镜 -> 视频 prompt -> 调用工具 -> notify_hunman 展示视频
```

## 输出契约

- **给视频模型**：本轮确认完成后才编译 `prompt` 并调用视频工具。有参考图或主 MD 路由来的 `ref` 中提取出的图片时同时传 `ref_images`，并在 prompt 开头用 `@image1/@product_ref` 声明职责；文本/视频/音频类 ref 先编译进 prompt 的参考素材说明、传播主线、创意视觉解法和六段式导演稿；`duration` 默认 15 秒，`ratio` 按投放平台和用户 prompt 推断。
- **确认前给用户**：只输出确认摘要或补充问题，不输出成片结果，不调用视频工具，不输出 `视频已生成`。
- **确认后给用户**：完整分镜表 + 视频结果；工具返回可用链接或本地路径后，按主 MD 要求调用 `notify_hunman` 工具展示视频，再使用成功格式。
- 默认不展示内部 prompt；用户要求调试、交接包或看 prompt 时才展示。
- 默认不输出字幕；广告大字不是字幕，必须遵守下文「字幕、广告语、台词符号」规则。

## 默认策略

- 当用户要求生成视频时，不要立即开始生成，也不要立即调用视频生成工具。必须先完成主 `SKILL.md` 的澄清确认流程。
- 未完成本轮确认时，只能输出确认摘要或补充问题；确认前不得调用视频工具，不得输出 `视频已生成`，也不得把上一轮的 `好/OK/嗯` 继承为本轮确认。
- 默认参数：本轮确认完成后，按确认后的 brief 生成，15 秒；比例先根据平台/投放场景推断，再进入确认摘要。抖音/小红书/快手/视频号/竖屏信息流默认 `9:16`；电视/TVC/Web 端/官网/YouTube/横版展播默认 `16:9`；电商详情/方形卡片可用 `1:1`；无法判断平台时再默认 `9:16`。镜头数遵守「高信息密度分镜矩阵」（常规 8-10 个关键镜头，强信息/快剪/功能/信息流路线 10-12 个关键镜头，极简留白路线也不得少于 7 个关键镜头）；不调用 imageedit，不默认生成分镜图片。
- 用户明确指定平台/比例/时长时优先遵从；用户未明说比例但 prompt 能推断平台时，必须写 `platform_ratio_decision`：平台/投放场景 -> 推荐比例 -> 推断依据。
- 有产品图时优先 `image_to_video`，产品图作为最高优先级参考；无可靠图片参考时用 `text_to_video`。
- 15 秒结构：`0-2s hook/蓄势`、`2-5s 产品揭示`、`5-10s 卖点证明`、`10-13s 结果/情绪价值`、`13-15s packshot hold`。
- 快剪可出现 0.5-1.5 秒节奏点，但每 2-3 个快切后必须给一次产品清晰锚点。
- 旁白密度按路线决定，并在确认摘要中展示完整旁白台词与结尾 slogan，以及 `voiceover_scene_map`：信息型/功能型/促销型路线默认 3-5 句主体旁白 + 1 句结尾 slogan；诗性/氛围/高端/香氛/珠宝/礼赠路线默认 2-3 句诗性主体旁白 + 1 句结尾 slogan。每句旁白必须落到 Hook/产品出现/卖点证明/使用结果/packshot 等画面阶段。
- 诗性旁白和 slogan 是固定搭配，不是可选关系；诗性旁白负责建立情绪线索、气味/质感/品牌记忆，slogan 负责最后收束。诗性路线不要求覆盖多数镜头，不逐镜头解释画面，避免香水、香氛、珠宝、高端礼盒等氛围大片旁白过密。
- 每条主体旁白都必须把产品资料和卖点转化为具体、可直接配音的台词，贴合产品卖点信息；禁止只写“高级旁白”“情绪旁白”“品牌解说”“突出品质”“强调功效”等抽象指令。
- 只有用户明确要求无旁白、纯音乐或纯鼓点时才取消旁白，并按用户意图设计 BGM、拟音和节奏。其他情况不能因为用户没提旁白、画面偏高端、ASMR、极简留白而自行取消。
- 快剪 + 动感音乐/鼓点不能自动视为无旁白；默认仍使用 3-5 句短促主体旁白 + 1 句 slogan。只有用户明确指定纯音乐/纯鼓点快剪时，才允许没有旁白台词。
- 大牌 TVC 通常在最后用一句简洁、可记忆、贴合品牌主张的 slogan 收尾；slogan 只能收束主体旁白，不能成为全片唯一一句台词。

## 传播主线与创意巧思

传播主线是关键参考信息，不是锁死视频模板。它只负责判断“这条视频优先让用户记住什么”，镜头设计仍要结合品类、调性、创意视觉法则和拍摄法则生成差异化方案。

执行关系：

```text
传播主线 = 决策锚点
创意巧思 = 视觉解法
品类/调性 = 执行语言
镜头设计 = 主线 + 创意法则 + 产品证明的综合结果
```

每条视频先写：

- `communication_mainline`：主传播目标，只选 1 个，如品质证明、功能卖点、促销转化、品牌理念、氛围种草、高端形象、使用场景、社媒停滑。
- `primary_message`：用户看完最该记住的一句话。
- `secondary_message`：辅助记忆点，只选 1 个。
- `avoid_message_drift`：明确不要跑偏的方向，如不要把品质片拍成纯促销、不要把品牌理念片拍成功能清单。
- `creative_visual_device`：本条视频选用的 1-2 个创意视觉解法。
- `visual_design_law`：图形、色彩、材质、构图等视觉组织法则。
- `shooting_design_law`：机位、运镜、景别、切点、声音先行等拍摄法则。

主线类型只提供倾向，不做硬绑定：

| 传播主线 | 重点 | 可选执行倾向 |
|---|---|---|
| 品质证明 | 材质、工艺、口感、肤感、结构 | 微距、慢推、材质证据、稳定 packshot；也可用快剪或图形化节奏提升记忆 |
| 功能卖点 | 效率、性能、体验提升 | 3+ 卖点拆解、操作闭环、前后结果；也可用高端光影或生活场景包装 |
| 促销转化 | 新品、优惠、限时、礼盒、爆品 | 强 hook、结果前置、短促 CTA、产品高频露出；避免变成低价叫卖 |
| 品牌理念 | 审美、价值观、生活方式 | 情绪叙事、视觉母题、人物/场景关系；仍要回到产品记忆点 |
| 氛围种草 | 场景向往、情绪、质感 | 空镜、自然光、材质联想、慢节奏；不能连续空镜忘记产品 |
| 高端形象 | 价格感、稀缺感、仪式感 | 延迟揭示、留白、低频声音、暗场显影；卖点表达要克制但清楚 |
| 使用场景 | 解决具体生活/工作场景 | 场景痛点 -> 产品进入 -> 使用动作 -> 结果状态；避免只拍环境 |
| 社媒停滑 | 先抓注意力 | 反差、声音钩子、硬切、视角反转；后续必须转入产品证明 |

创意视觉解法可跨主线自由组合：

| 创意法则 | 用法 |
|---|---|
| 形状母题 | 用圆形、弧线、切面、网格、对称、重复形状贯穿产品、道具和转场 |
| 色彩递进 | 从低饱和到品牌色、从暗到亮、从冷到暖，形成视觉推进 |
| 材质联想 | 用水、雾、丝绸、金属、玻璃、颗粒、泡沫、光束表达真实卖点或质感 |
| 尺度反差 | 微距细节与完整产品交替，建立高级感和产品记忆 |
| 动作重复 | 开、按、倒、抹、戴、合等产品动作重复 2-3 次形成记忆点 |
| 对称构图 | 中轴、镜像、几何排列、整齐陈列，适合高端、科技、礼盒和美妆 |
| 前景遮挡 | 用叶片、布料、手部、包装边缘、液体或光影擦镜转场 |
| 声音先行 | 先听到开盖、倒入、按键、布料、冰块、喷洒声，再切到产品动作 |
| 匹配剪辑 | 用同形、同色、同方向运动连接原料、产品、使用动作和结果 |
| 负空间留白 | 用空白、静止、慢动作、低频声音制造价格感和呼吸感 |
| 节拍图形化 | 用光带、色块、分屏、产品轨迹、几何运动承接 BGM 节奏 |
| 视角反转 | 从用户视角、产品视角、俯拍视角、过肩视角或反射视角切入 |

组合示例：

```text
communication_mainline: 品质证明
primary_message: 这款产品的真实材质和细节质感值得被看见
secondary_message: 使用动作带来可感知的高级体验
creative_visual_device: 尺度反差 + 材质联想
visual_design_law: 微距纹理与完整产品交替，暖金高光贯穿
shooting_design_law: 微距巡航 + 焦点转移 + 稳定 packshot
avoid_message_drift: 不拍成纯促销，不用大量广告大字替代质感证明
```

## Seedance 2.0 提示词规范

遵循 Seedance 2.0 的提示词写法：先锁主体，再按镜头时序组织，最后用画质、风格和约束词收紧。

- 进阶公式：`精准主体 + 动作细节 + 场景环境 + 光影色调 + 镜头运镜 + 视觉风格 + 画质 + 约束条件`。
- 给用户看的分镜可以保留 15 秒时间段；给 Seedance 的 prompt 编译时用 `镜头1`、`镜头2`、`镜头3`、`镜头4` 顺序组织，不强行把每个镜头写死为精确秒数。15 秒由工具参数 `duration=15` 控制。
- 每个镜头按固定顺序写：`运镜或镜头切换方式 -> 主体动作与表情/状态 -> 位置或空间变化 -> 音频信息`。
- 每个镜头尽量只指定 1 种主要运镜方式；不要在同一镜头同时堆叠推、拉、摇、移、环绕，避免画面不稳定。
- 动作要具体到手、头部、肩背、产品接触点、力度、速度、幅度；优先低缓、连续、小幅动作，并补充前后动作的过渡衔接。
- 情绪不要只写“高级、开心、惊艳”；要外化成可见动作，如嘴角上扬、手指轻按、肩膀放松、呼吸停顿、眼神停留。
- 有参考素材时先定义主体：`将图片1中的产品定义为产品主体`；后续每次涉及产品都使用同一标签，如 `产品主体@图片1`。不要直接用 Asset ID 替代 `图片1/视频1/音频1`。
- 多模态参考按维度写：`参考视频1中的运镜/动作节奏/光影风格/音效`，`参考音频1中的音色/节奏/氛围`；不要复制参考视频的品牌、字幕、口号或专属资产。

### 视频改写 Prompt 细化策略

当用户提供参考视频、截图式 prompt、已有视频文案或要求“参考这个改写”时，不是只改文案，而是把参考拆成可迁移的视听语言，再重编译成产品广告 prompt。

改写前先提取：

- `reference_structure`：参考视频的段落结构，如 selfie 开场、微距证明、静态展示、packshot 收尾。
- `reference_pov`：机位和拍摄方式，如手机手持、三脚架定机位、俯拍、微距、低机位、硬切。
- `reference_motion`：动作节奏，如产品从画面边缘进入、手部按压、轻微回正、镜头完全静止、一次明确指向动作。
- `reference_emotion_arc`：情绪弧线，如惊喜、认真展示、结果满足、强 CTA；产品广告可转译为“期待 -> 证明 -> 收益 -> 记忆点”。
- `reference_audio_logic`：声音关系，如 iPhone 麦克风质感、房间底噪、强拟音、节拍落点、旁白能量。
- `reference_risk`：需要剥离的内容，如原品牌、人物身份、字幕排版、画面文字、水印、专属 IP、不可复用台词。

改写模式只选 1 个主模式：

| 模式 | 适用 | 执行 |
|---|---|---|
| 保结构换产品 | 参考视频结构强、产品不同 | 保留段落顺序和镜头关系，替换产品动作、卖点证明和场景道具 |
| 保调性换场景 | 参考视频氛围好但场景不适合 | 保留光线、节奏、情绪能量，重建符合品类的广告场景 |
| 保运镜换卖点 | 参考视频镜头语言好 | 保留手持/定机位/微距/硬切等机位逻辑，把每个镜头改为产品卖点证明 |
| 全量广告化重写 | 参考视频只提供方向 | 只取高层策略，重新生成调性、分镜、音频和产品证明链 |

改写输出必须使用六段式导演稿：

```text
Style & Mood: 拍摄质感 + 光线 + 机位组合 + 剪辑方式 + 平台比例。
Narrative Summary: 15 秒完整广告弧线，用一句话说明产品如何从 hook 到证明再到 packshot。
Dynamic Description: 按 Cut/镜头逐段写动态动作、产品位置、手部接触点、表情/状态、运镜、转场和声音触发。
Static Description: 固定空间、背景、光源、道具和产品陈列规则，保证镜头之间空间不漂移。
Audio: BGM/人声/拟音设计，沿用现有符号规则：音乐用()，拟音用<>，台词/口播用{}。
Constraints: 产品一致性、无字幕、无水印、不复制参考视频品牌资产和画面文字。
```

`Dynamic Description` 每个 Cut 必须写清：

- 景别和机位：如 `TIGHT CLOSE-UP HANDHELD`、`MACRO TRIPOD`、`TOP-DOWN STATIC`、`MEDIUM PACKSHOT`。
- 产品控制：产品数量、朝向、logo/标签是否正对镜头、占画面比例、是否停稳、手指是否遮挡。
- 主动作：只写一个主动作，如拿起、打开、按压、倒入、涂抹、轻压、展示、指向、放回。
- 微动作：允许 1-3 个服务广告情绪的小动作，如眼神停留、手腕微调、肩膀放松、短暂停顿、轻笑、呼吸停住。
- 机位状态：手持微晃、三脚架完全静止、微距固定、硬切、匹配剪辑、前景遮挡擦镜。
- 转场提示：每段结尾写 `Hard cut to`、`Match cut to`、`Foreground wipe to` 或声音先行转场。

保留规则：

- 只提取参考视频的结构、机位、节奏、情绪弧线、声音关系；不复制品牌、字幕、画面文字、水印、人物身份或专属视觉资产。
- 原参考视频有字幕时，只提取信息节奏，不复刻字幕形式；默认仍保持无字幕。
- 原参考视频是 UGC 风格时，可以迁移手机感、手持微晃、自然光和口播能量，但产品营销片仍必须完成产品锁定、卖点证明和 packshot。
- 原参考视频是强情绪口播时，可以迁移情绪强度，但要按产品调性降噪：高端/医研/科技路线不使用过度尖叫、夸张表情或低质网感。

### 字幕、广告语、台词符号

Seedance prompt 内部按信息类型使用固定符号，避免模型误解。

| 信息类型 | 符号 | 写法 |
|---|---|---|
| 音乐/BGM | `()` | `(背景中播放低频弦乐与轻电子氛围，80-100 BPM)` |
| 音效/拟音 | `< >` | `<开盖声清脆短促>`、`<液体倒入玻璃杯的气泡声>` |
| 台词/人物口播 | `{}` | `{这一口，很清爽}`；小语种需写 `用日语说道{こんにちは}` |
| 字幕 | `【】` | 默认禁用；只有用户强制要求字幕时才用 `【...】` |
| 广告大字/标题字 | 中文双引号 | `“真香”`、`“轻到忘记”`；它不是字幕，不使用 `【】` |

默认字幕策略：

- 默认写入约束：`保持无字幕，避免生成任何文字或字幕，不生成 caption、贴纸字、水印、乱码`。
- 允许广告大字时，必须同时写清 `文字内容、颜色、字体风格、出现方式、出现时机、出现位置`，并确认不遮挡产品 logo、包装关键信息、手部证明动作和人物眼睛。
- 台词语言必须统一，避免中英文混用；品牌名、SKU、专有名词除外。
- 旁白、人声、人物台词都属于音频信息，不等于画面字幕；不要因为有旁白而生成字幕。

## 参考资产职责

内部先写 `reference_asset_roles`：

- 产品图：锁包装、logo、SKU、材质、比例、颜色、结构、标签布局，不能被 AI 重绘替代。
- 人物图：锁人物身份、服装、妆发、体态；只作为演员/手模/模特参考，不改变产品。
- 场景图：锁空间、材质、光线、陈列关系。
- 风格图：只参考色调、光影、构图和节奏，不改变产品结构与品牌元素。

### 参考图使用判断

用户上传参考图时，必须先判断参考图职责，不要默认把整张图的场景、道具、构图都照搬。

写清 `reference_usage_decision`：

- `只提取关键产品视觉`：商品图、白底图、电商图、普通截图；只锁产品外形、结构、材质、颜色、logo、SKU、标签布局，场景重新设计。
- `同时参考产品 + 场景`：用户明确要求参考场景，或原图空间/光线/陈列与调性强相关；场景只提取材质、光线、陈列关系，不抢产品主角。
- `只参考风格/调性`：参考图不是该产品或用户只要风格；只提取色调、光影、构图、景深、节奏。
- `不参考图中场景`：低质截图、社媒界面、聊天截图、杂乱生活照、水印/字幕/无关人物；只提取可用产品视觉。

prompt 中必须写清：`reference_asset_roles` 和 `reference_usage_decision`，例如：`图片1只作为产品视觉参考，不参考原图背景和桌面；广告场景另设为冷白科技浴室台面`。

产品图进入 prompt 首段必须写：

```text
@image1 作为唯一产品参考，严格保持产品数量、外形结构、尺寸比例、颜色、材质质感、反光、透明度、表面纹理、logo/品牌标识位置、标签形状、主文字布局、SKU 和包装细节一致；不要重绘包装，不要换颜色，不要生成新 logo/新标签/假文字。
```

## 产品一致性最高优先级

- 生成图好看但产品不准也必须丢弃；产品准确优先于氛围、人物和场景。
- 每个涉及产品的镜头都重复产品锁定：产品位置、朝向、占画面比例、logo 是否正对镜头、手部是否遮挡标签。
- 产品至少清晰出现 3 次：开场 hook、卖点证明特写、结尾 packshot。
- 跨镜头产品尺寸比例恒定，用手掌、桌面、杯具、模特身体或厨房器具做比例锚点。
- 反光、阴影、接触点匹配同一光源；禁止悬浮、瞬移、贴图感、包装漂移。
- 真实品牌包装、SKU、门店招牌和标签默认禁止 AI 重绘，只引用用户原始产品图。

## 通用优质广告策略

这些策略先于品类执行；不要照搬参考视频里的具体品牌、文字或画面，而是抽象成可迁移的镜头语法。

| 策略 | 作用 | 执行规则 |
|---|---|---|
| 延迟揭示 | 建立高级感和期待 | 不急着全露产品；先用暗场、局部轮廓、品牌色、材质、手部动作或场景细节蓄势，再进入完整 hero shot |
| 材质隐喻 | 把卖点变成可见感受 | 用水滴、雾气、玻璃折射、金属高光、织物波纹、颗粒、光束、流体、机械结构等表达“清爽、精密、柔软、速度、能量、质感”；隐喻必须服务真实卖点，不编造功效 |
| 模块快剪 | 提升信息密度和节奏 | 用分屏、拼贴、色块、几何轨迹、产品多尺度重复、拆解视角快速组织信息；若需要广告大字，只能出现 1-3 次，每次 2-8 个中文字符，且必须写成 `“...”` |
| 证明闭环 | 让广告不是空镜头 | 每条片至少完成一次“产品出现 -> 人/手/环境交互 -> 可见证明 -> 结果状态 -> 产品记忆点” |
| 视觉母题 | 增强品牌记忆 | 选择一个母题贯穿 3 次以上：主色、圆形/弧线/切面、光束方向、液体运动、材质纹理、手势或陈列方式 |
| 对比转化 | 强化利益点 | 用前后状态、旧方案 vs 新方案、静态 vs 动态、暗到亮、杂乱到有序、粗糙到精致表达收益；不要捏造无法证明的数据 |
| 结尾锚定 | 留下产品记忆 | 最后 1-2 秒稳定 packshot，产品正面、logo/包装可见、光线收束，BGM 或拟音落点同步停稳 |

## 优质素材归纳策略

从参考广告中只提炼通用策略，不复制具体品牌、产品、字幕、口号或专属画面。

| 策略 | 适用 | 执行规则 |
|---|---|---|
| 暗场预热到产品显影 | 高端、美妆、香氛、礼盒、3C | 先用黑场、低照度轮廓、局部高光或材质运动蓄势，再让产品从暗到亮进入 hero；适合建立价格感 |
| 材质世界围绕产品 | 食品饮品、美妆、香氛、服装、家居 | 让液体、雾气、丝绸、玻璃、金属、叶片、颗粒、光束围绕产品运动，材质既是氛围也是卖点隐喻 |
| 产品结构拆段记忆 | 包装有辨识度的产品 | 按“外形轮廓 -> 关键材质 -> logo/标签 -> 完整产品”拆成 3-4 个记忆点，避免只给一个远景 |
| 感官证据链 | 食品、饮品、美妆、面料、香氛 | 用连续可见证据表达体验：冷雾/水珠/倒入/泡沫/拉丝/涂抹/吸收/布料摩擦；每个证据要接回产品 |
| 手部仪式感 | 美妆、香氛、礼赠、厨具、3C、配饰 | 手部动作只做“触碰、拿起、打开、递送、使用、托举”，动作慢而准；手不能遮挡 logo 和关键结构 |
| 源头元素转场 | 食品饮品、美妆、香氛、天然原料产品 | 从花、叶、水、谷物、果肉、矿物、织物等源头元素过渡到产品，说明质感来源，但不编造功效 |
| 生活结果落地 | 厨具、家居、服装、3C、日用 | 不能只有空镜和材质；必须给一次人/手/环境使用后的结果状态，让利益点落到真实场景 |
| 信息图形转译 | 3C、办公、功能型产品、年轻化快剪 | 少用文字堆参数；用几何轨迹、分屏、产品多尺度、灯光扫描、节拍切换表达功能层级 |
| 终局包装锚定 | 所有品类 | 最后回到稳定完整产品，正面、比例、logo/包装清楚，声音落点同步停稳；不要在最后一秒加入新信息 |

## 短广告爆点机制

15 秒广告优先选择 1 个主爆点机制，不要同时堆满。爆点机制必须让产品成为转折核心。

| 机制 | 适合场景 | 执行规则 |
|---|---|---|
| 格式劫持型 | 信息流、平台内广告、年轻化品牌 | 画面一开始像普通使用场景、系统界面、直播间、教程或剧情片段，1-2 秒内反转为产品广告；不得仿冒真实平台 UI 或生成误导性按钮 |
| 人物反差型 | 有代言人、达人、手模、强人物设定 | 用人物身份/语气/动作与产品形成反差，如严肃人物做轻松动作、专业人士被简单功能打动；人物不抢产品主角 |
| 产品动作仪式型 | 有明确使用动作的产品 | 把一个动作拍成记忆点：扭开、按下、涂上、咬开、戴上、放入、拉开、合上；动作必须出现 2 次以上并形成视觉母题 |
| 夸张后果型 | 食品、饮品、美妆、家居、日用、工具 | 产品使用后触发夸张但可理解的视觉后果，如表情变化、空间变亮、物体有序、节奏突然加快；夸张服务利益点，不编造功效 |
| 品类错位型 | 低关注度品类、需要破圈的产品 | 用不属于该品类的拍法制造新鲜感，如把日用品拍成大片、把工具拍成潮流单品、把食品拍成仪式感场景；产品功能仍要讲清 |
| 单功能证明型 | 3C、工具、小家电、清洁、效率类产品 | 只讲一个功能，用 2-3 个镜头连续证明：问题 -> 操作 -> 结果；不要堆参数 |
| 声音钩子型 | 有强动作声、口头禅、节奏点的产品 | 用开盖声、脆响、按键、倒入、摩擦、短促人声或音乐 sudden stop 做记忆点；声音必须和画面动作同步 |

广告大字使用规则：

- 大字不是字幕，不逐句复述旁白；只做 hook、转折、卖点锚点或结尾记忆点。
- 单条 15 秒最多 3 次大字；每次停留 0.4-1.2 秒。
- 大字必须使用中文双引号包裹，格式如 `“真香”`、`“秒变顺手”`、`“这一口”`；它属于广告语/标题字，不使用字幕符号 `【】`。禁止英文引号、无引号、乱码、长句、连续滚动字幕。
- 每次大字必须写全 6 项：文字内容、颜色、字体风格、出现方式、出现时机、出现位置。
- 大字不得覆盖产品 logo、包装关键信息、人物眼睛、手部证明动作；优先放在留白、背景虚化区、画面边缘或独立转场帧。
- 字体风格必须适配品类、调性和画面材质，不能全部使用黑体；每次大字都写清字体气质、字重、字距、颜色、描边/阴影/发光和与背景的层级关系。
- 出场形式必须有设计，不能只写“出现文字”；从直接出现、渐进出现、跳动出现、切片滑入、跟随产品轨迹、随节拍弹出、从材质中浮现、光扫显影、镜头遮挡转场中显现里选择。


广告大字字体与动效矩阵：

| 调性 | 字体风格 | 出场形式 | 避免 |
|---|---|---|---|
| 高端大气 | 细衬线、现代宋、轻奢窄体、低对比优雅字形；金色/白色/低饱和色，字距略宽 | 渐进出现、光扫显影、从暗场中淡入、随镜头慢推浮现 | 粗黑体、弹跳、强描边、廉价发光 |
| 快剪情绪 | 粗窄体、压缩黑体、运动标题字、轻微倾斜；高对比色或品牌主色 | 跳动出现、节拍弹出、切片滑入、撞击缩放 | 长停留、过细字重、无节奏淡入 |
| 真实生活 | 圆润无衬线、手写感但清晰、亲和中粗字；暖白/柔和品牌色 | 直接出现、轻微上浮、手部遮挡转场后出现 | 过度潮流、强烈霓虹、难读手写 |
| 科技发布 | 几何无衬线、等宽感、细中黑、数字界面感；冷白/银灰/品牌冷色 | 线性扫描、模块展开、跟随产品轨迹、淡入锁定 | 复古手写、花体、过多装饰 |
| 质地感官 | 柔和圆体、细宋/优雅衬线、低对比字形；与材质高光同色系 | 从液体/雾气/颗粒/布料纹理中浮现，慢速渐入 | 硬切大字、强节拍弹跳、厚重黑块 |
| 功能证明 | 清晰中黑无衬线、信息标签感、字重稳定；高可读对比 | 直接出现、步骤式显现、跟随证明动作出现 | 花体、过度变形、遮挡操作动作 |
| 节日礼赠 | 温暖衬线、圆润标题字、轻手写点缀；金色/红色/暖白克制使用 | 暖光淡入、礼盒开合转场显现、粒子轻闪 | 廉价闪烁、满屏花字、过多颜色 |

## 运镜策略矩阵

15 秒广告必须写清每段镜头的运动路径、速度、焦点起落和切换方式。不要只写“高级运镜”或“慢镜头”。

| 运镜策略 | 作用 | 执行规则 |
|---|---|---|
| 中心慢推揭示 | 建立 hero 感 | 产品在画面中心或黄金位保持稳定，镜头从环境/材质慢推到产品；适合开场和结尾前回归 |
| 微距巡航 | 放大材质和工艺 | 镜头沿瓶盖、标签、切面、屏幕、面料、食物断面或器具边缘横移/上移，最后焦点落到 logo 或核心卖点 |
| 半环绕视差 | 增加空间和大片感 | 产品基本不变形，镜头小幅环绕 15-45 度，前景材质与背景形成视差；禁止让包装文字扭曲漂移 |
| 前景遮挡擦镜 | 做自然转场 | 用手、布料、叶片、液体、器具、包装边缘从镜头前经过，完成切镜或景别变化；遮挡后产品位置要连续 |
| 焦点转移 | 引导注意力 | 从水滴、纹理、手指、原料、气泡、按键等前景对焦，拉焦到产品 logo/使用动作/结果状态 |
| 材质运动驱动 | 让画面自己动起来 | 丝绸波纹、液体倒入、泡沫上升、水波扩散、叶片漂浮、蒸汽升腾、光束扫过作为节奏点和转场，不硬切堆镜头 |
| 手部跟随 | 强化真实使用 | 镜头跟随手部进入、触碰、打开、涂抹、倒入、拿起、放下；动作必须有接触点、重量感和停稳 |
| 证据链切换 | 连续证明卖点 | 按“动作开始 -> 质地变化 -> 结果出现 -> 产品回归”切 3-4 个景别；适合饮品、食物、美妆、清洁、厨具 |
| 源头到产品转场 | 建立成分/质感联想 | 从原料、自然元素、材质纹理转到产品包装；只表达来源感和情绪，不把素材说成真实配方证明 |
| 节拍快切定格 | 提升短广告爽感 | 快剪也要有定格锚点：每 2-3 个快切后给 0.5 秒产品清晰画面，避免用户只记住氛围不记产品 |

运镜组合规则：

- 每条 15 秒至少使用 2 种真实运镜策略，且至少 1 次明确回到稳定产品 packshot。
- 每个分镜都要写 `camera_path`、`camera_speed`、`focus_plan`、`transition_design`；快剪路线也不能省略。
- 区分真实运镜和数字变焦：优先写推轨、横移、俯仰、半环绕、跟随、拉焦、遮挡擦镜；不要全靠 zoom in/zoom out。
- 运镜必须服务卖点：质地用微距/拉焦，功能用跟随/证据链，礼赠用慢推/托举，科技用推轨/光扫，快剪用节拍切换/短定格。

## 速度节奏与变速策略

每条 15 秒至少包含 1 次速度变化设计，不得全片同速。速度变化必须服务产品信息、卖点证明、情绪停顿或声音落点，不能为了炫技让产品变形、logo 漂移或动作不连续。

| 速度策略 | 适用位置 | 执行规则 |
|---|---|---|
| 慢动作 | 冰块飞溅、液体倒入、膏体推开、布料/金属/食物质地、产品 hero 揭示 | 用于放大质地和价值感；镜头必须有清晰焦点，不能只是“慢慢推近” |
| 中速真实动作 | 手部打开、拿起、倒入、涂抹、按键、佩戴、使用 | 保留真实重量感和接触点，适合证明动作闭环 |
| 节拍快切 | 信息流、功能拆解、规格/SKU、卖点连打、年轻化快剪 | 每 2-3 个快切给一次 0.4-0.8s 产品清晰定格锚点 |
| 加速蒙太奇 | 多场景、多规格、多步骤、多卖点压缩展示 | 用于提升信息密度；必须按同一逻辑排列，不能乱堆素材 |
| 先慢后快 | 开场氛围蓄势后进入卖点证明或快剪 | 适合高端揭示、夏日感官、科技发布，从质感转入信息 |
| 先快后慢 | 先用强钩子/快切停滑，再落到产品 hero 或 packshot | 适合信息流和爆点机制，最后必须给产品稳定记忆点 |
| speed ramp/变速推进 | 开盖、倒入、水花、旋转、抛接、布料展开、灯光扫过等动作爆点 | 在动作峰值前后变速，落点必须同步 BGM/拟音；不得让产品形态跳变 |
| 定格锚点 | 产品正面、logo、规格、结果状态、最终 packshot | 每段高密度信息后给观众识别时间；最后 1 秒必须稳定 hold |

组合规则：

- 高端/轻奢：慢动作 + 中速真实动作 + 定格锚点，少用连续加速。
- 快剪/信息流：节拍快切 + 加速蒙太奇 + 先快后慢 + 定格锚点。
- 饮品/食品/质地感官：慢动作 + speed ramp + 中速真实动作，突出水、冰、气泡、倒入、入口反应。
- 科技/功能：中速真实动作 + 加速蒙太奇 + 先慢后快，强调结构、响应、步骤和结果。
- 自然/疗愈/国风：慢动作 + 先慢后快或先快后慢的轻微变化，避免过度快切破坏氛围。


## 15s 通用结构模板

先选一个结构模板，再按品类填充动作。15 秒营销片默认采用高信息密度分镜：用 8-12 个关键镜头综合产品、卖点、场景、使用、声音和收束信息；参考 12 格分镜只作为信息密度参考，不默认生成分镜图片。

- **高端揭示型**：0-2s 暗场/材质/轮廓蓄势；2-5s 产品 hero；5-9s 材质或核心卖点微距；9-12s 使用/结果；12-15s packshot。适合需要质感、价格感、礼赠感的产品。
- **模块快剪型**：0-2s 强视觉 hook；2-5s 产品多角度/多尺度重复；5-9s 功能或场景模块切换；9-12s 使用结果；12-15s 产品记忆点。适合年轻化、功能点多、信息流广告。
- **证明转化型**：0-2s 痛点/场景；2-5s 产品进入；5-10s 动作证明；10-13s 前后结果；13-15s packshot。适合强调效率、效果、体验差异的产品。
- **感官沉浸型**：0-3s 超近材质/声音 hook；3-6s 产品局部揭示；6-10s 触感/味觉/质地/动作证明；10-13s 情绪反馈；13-15s packshot。适合靠质地、口感、手感、香气打动用户的产品。
- **爆点机制型**：0-2s 格式劫持/反差/声音钩子；2-6s 产品动作仪式或单功能证明；6-11s 夸张后果/对比转化；11-13s 产品正面回归；13-15s 大字记忆点 + packshot。适合需要强传播和高停留的短广告。

## 调性先行执行策略

先确定调性，再结合品类生成最终视听语言。不要只按品类套模板：同一个饮品可以是夏日感官、小清新自然或快剪情绪；同一个 3C 可以是 Apple 式极简、小米式科技生活或高端机械工业。

执行顺序：

```text
调性路线 -> 品类证明动作 -> 镜头密度 -> 氛围镜头比例 -> 产品露出节奏 -> 运镜/转场 -> BGM/人声/拟音 -> 分镜脚本板
```

调性决定：

- 镜头数量：默认不得少于 8 个关键镜头；常规 8-10 个关键镜头，强信息/快剪/功能/信息流路线 10-12 个关键镜头；极简高端、纯 ASMR 或强留白路线也至少 7 个关键镜头，并用更短更克制的镜头承担信息，而不是压成 3-6 镜。
- 氛围镜头预算：高端仪式 25%-40%，小清新自然 40%-60%，森林疗愈 45%-65%，科技发布 10%-25%，快剪情绪 10%-25%。
- 产品露出节奏：科技/功能先露产品，氛围/自然先露环境，高端仪式先露道具和动作，快剪先露冲突或结果。
- 声音结构：科技用合成器与低频脉冲，自然用环境声和轻音乐，快剪用鼓点，仪式用克制弦乐/钢琴，感官用拟音优先。

## 高信息密度分镜矩阵

无分镜版本不调用 imageedit，也不默认生成分镜图片，但视频 prompt 仍必须使用高信息密度分镜。默认不得少于 8 个关键镜头；常规 8-10 个关键镜头，强信息/快剪/功能/信息流路线 10-12 个关键镜头。参考 12 格分镜的密度时，只吸收“信息层级和镜头覆盖”，不要输出图片。

12 格参考密度用于综合所有视频信息：产品外观、冰感/质地/材质、卖点证明、动态环境、使用场景、人物反馈、规格/SKU、品牌收束和声音落点。实际生成可按产品复杂度删减为 8-10 镜，但不能删掉产品锁定、细节证明、使用/结果和最终 packshot。

| 镜头 | 必须差异 | 常用景别/角度 | 画面功能 |
|---|---|---|---|
| 01 Hook/世界观建立 | 与后续产品特写不同，先给冲突、材质、环境或结果钩子 | 远景/超近微距/低照度轮廓/前景遮挡 | 抓注意力，建立调性和视觉母题 |
| 02 感官元素特写 | 用冰、水、叶片、质地、机械部件、材质纹理等建立感官证据 | 超近微距、慢动作、俯拍、拉焦 | 先给“为什么想看下去”的感官理由 |
| 03 产品登场/hero | 产品第一次清晰出现，logo/包装可识别 | 中近景、正面或 3/4 角度、低机位或平视 | 锁定产品主体和品牌记忆 |
| 04 产品细节/标签/材质 | 必须换成微距或局部，不重复完整 packshot | 微距、斜侧近景、局部切面、焦点转移 | 证明材质、包装、结构、标签或工艺 |
| 05 卖点信息证明 | 用图形化信息、动作或旁白承载核心卖点，不默认字幕 | 局部 + 元素组合、分屏、图形轨迹、轻信息画面 | 把卖点转成可理解的证据 |
| 06 动态环境/产品运动 | 产品与环境元素形成旋转、环绕、流动、切换或节拍动作 | 环绕、中景、推轨、跟拍、硬切 | 提升节奏，避免全片静物堆叠 |
| 07 原料/质地/结构证明 | 展示茶叶、膏体、金属、织物、食材、部件等来源或质地 | 超近微距、慢动作、俯拍、侧逆光 | 让品质证明更具体 |
| 08 使用引入/人物或手部 | 必须出现手、人、工具或环境交互 | 手部跟随、侧面近景、过肩视角、桌面俯拍 | 证明产品如何进入真实场景 |
| 09 使用结果/情绪反馈 | 展示使用后的状态、人群反馈或空间变化 | 半身、场景中景、生活环境、结果对比 | 把功能翻译成用户收益 |
| 10 产品静物/高级定格 | 产品稳定停留，强化包装、材质、光影和价值感 | 正面定机位、俯拍、桌面陈列、浅景深 | 给观众清晰记住产品的时间 |
| 11 规格/SKU/组合信息 | 如有规格、口味、系列、套装、容量，必须清楚组织 | 排列、俯拍、横移、分层陈列 | 承载购买前关键信息 |
| 12 最终 packshot/品牌收束 | 必须稳定、完整、可收尾，与 03/10 构图不同 | 正面定机位、礼盒陈列、产品 + 品牌记忆点 | 留下产品记忆，最后 1 秒 hold |

差异检查：

- 相邻两镜不能同时使用相同景别、相同相机角度、相同产品位置和相同动作。
- 8-12 个镜头里至少包含 4 种景别：微距/近景/中景/全景/俯拍/低机位任选。
- 8-12 个镜头里至少包含 3 种主体关系：产品单独、手部交互、人物/场景、材质元素、结果状态、规格组合任选。
- 8-12 个镜头里至少包含 3 种构图方向：正面居中、左/右偏置、俯拍、低机位、前景遮挡、半环绕起点任选。
- 如果是氛围型路线，至少 3 个氛围或源头元素镜头，但每 2-3 个镜头必须回到产品、使用动作或卖点证明。
- 如果是快剪/抖音路线，10-12 个镜头中至少 6 个必须是不同动作或不同证明点，不能只换色块和文字。

## 源头元素闭环

适合食品、饮品、美妆、香氛、茶饮、天然原料产品。结构是：`源头元素 -> 氛围空间 -> 产品出现 -> 质地/使用证明 -> 回到源头或标签`。

- 食品/饮品：果实、冰、水、气泡、谷物、蒸汽先行，再进入包装和入口反应。
- 美妆/香氛：花、梨、柑橘、木质、雾气、布料先行，再进入瓶身、喷洒、肌肤/空气反馈。
- 茶饮/自然类：树影、溪流、叶片、泡泡、苔藓先行，再进入杯体、原料、饮用场景。
- 不把源头元素说成真实配方证明，除非用户提供明确产品资料；只作为视觉联想和情绪来源。

## TVC 场景连续性

TVC/广告大片通常应保持相对一致的主场景、光线逻辑和视觉世界观，避免为了信息密度把镜头剪成互不相关的场景混剪。

可以加入原料、质地、使用场景、情绪反馈等辅助镜头，但辅助场景应与主场景有明确关联，例如由产品动作、材质元素、光影、声音落点或人物动线自然过渡。

如果创意本身需要多场景快剪，可以使用多场景结构，但必须保持统一调性、统一产品锚点和清晰转场因果。

## 氛围空镜策略

氛围型路线可以加入同类型元素空镜头，但空镜必须服务调性和产品联想，不能变成无产品的风景混剪。

- 15 秒内可安排 1-2 个空镜头；氛围预算高的路线最多 2 个。
- 空镜必须与产品同类型元素相关：饮品用水/冰/果皮/气泡，香氛用花影/织物/雾气/木质，护肤用水珠/柔光/肌肤纹理，茶/国风用茶席/窗影/器皿，科技用光带/材质纹理/桌面秩序。
- 空镜之后必须回到产品、使用动作或卖点证明，不能连续空镜超过 2 个。
- 空镜也要写 `shot_goal`，如氛围建立、材质隐喻、转场铺垫、情绪停顿。
- 空镜不允许出现与产品品类冲突的元素，如护肤面霜出现可食用果肉入口、科技产品出现无关自然风景、香氛产品变成纯花海 MV。
- 常用衔接：同形/同色匹配剪辑、光影扫过、手部动作前停顿、声音先行后切产品动作。

## 高端仪式场策略

把产品放进“被认真对待”的仪式场，提升价格感和记忆点。可迁移到香氛、美妆、礼盒、珠宝、服装、厨具、高端食品。

- 常见仪式道具：手套、银盘、餐铃、礼盒、绒布、聚光、餐桌、幕布、展示台、收藏柜。
- 镜头顺序：仪式动作开场 -> 产品被呈现 -> 材质/标签微距 -> 使用或结果 -> 俯拍/正面 packshot。
- 声音：少音乐、重拟音，如 `<餐铃轻响>`、`<瓶盖轻触银盘>`、`<布料轻擦>`；人声少词低声线。
- 避免：道具抢产品、仪式感与价格带不匹配、廉价闪光、过度戏剧化。

## 护肤/医研专项策略

适合面霜、精华、乳液、防晒、洁面、修护霜、抗老紧致、敏感肌、屏障护理、痘肌护理。护肤广告的核心不是“拍得像奢侈品”，而是建立可信证据链：`皮肤需求 -> 产品锁定 -> 成分/质地 -> 使用动作 -> 吸收或肤感结果 -> packshot`。

### 品类识别优先级

如果产品图、包装或 brief 出现以下信号，优先判定为 `功效护肤/医研护肤`，不要因为原料名误判成食品或香氛：

- 产品词：cream、serum、moisturizer、lotion、cleanser、sunscreen、SPF、balm、essence、toner、eye cream、面霜、精华、乳液、防晒、洁面、修护霜、眼霜。
- 功效词：firming、repair、hydrating、anti-aging、brightening、sensitive、barrier、non-comedogenic、fragrance-free、紧致、修护、保湿、抗老、提亮、敏感肌、屏障、低刺激、不闷痘、无香精。
- 成分词：ceramide、hyaluronic acid、niacinamide、retinol、peptide、vitamin C、glycerin、AHA/BHA、神经酰胺、玻尿酸、烟酰胺、视黄醇、胜肽、维 C、甘油、酸类。
- 专业词：dermatologist、clinical、lab tested、皮肤科、临床、医研、实验室、专业护理。

若护肤品名称含 black tea、peach、mango、citrus、rose 等原料名，只能作为成分联想、色彩和质地隐喻；不要生成可食用果肉、果干切片、饮品倒入等食品动作。例：`BLACK TEA FIRMING CORSET CREAM` 应判定为紧致面霜，黑茶只可转译为暖棕茶色、轻微茶叶影、琥珀高光，不能拍成芒果干或茶饮。

### 15s 护肤证据链

| 时间段 | 镜头目标 | 执行规则 |
|---|---|---|
| 0-2s | 需求 hook | 可用干燥纹理、暗淡肤感、早晚护肤场景、瓶盖开启声；不要制造夸张病症或医美前后对比 |
| 2-4s | 产品锚定 | 产品正面 hero，logo/SKU 清楚；如果是真实品牌包装，标签不能重绘或漂移 |
| 4-7s | 质地证明 | 刮勺取膏、滴管滴落、指腹推开、膏体拉峰、轻薄成膜；镜头必须能看出质地差异 |
| 7-10s | 使用动作 | 手背、脸颊、颈部或下颌线轻推按摩；动作慢、力度轻、接触点明确，不遮挡产品 |

| 12-15s | packshot | 产品回到梳妆台/浴室台面/专业白色实验台，光线稳定，最后 1 秒 hold |

### 护肤广告禁区

- 不把护肤品拍成食品：禁止果肉入口、饮品倒入、咀嚼、拉丝食物感、食用级摆盘。
- 不把面霜拍成香氛：香氛式花瓣雾气可以少量做氛围，但必须回到取用、推开、吸收、肤感证明。
- 不编造医疗功效：禁止治愈、消炎、修复疾病、逆转年龄、永久淡纹；没有用户资料时不写具体百分比和临床数据。
- 不做夸张前后脸：避免一秒变脸、毛孔凭空消失、皱纹完全抹平；只写“看起来更细腻/更润泽/更有光泽”。
- 不过度磨皮：保留真实肌肤纹理，妆效/肤感自然，不生成塑料皮。

## 音频策略

- BGM 先跟结构模板，再跟品类，并在 prompt 中用 `()` 包裹：高端揭示型用低频氛围/弦乐/钢琴；模块快剪型用鼓点/电子/funk；证明转化型用稳定节拍；感官沉浸型用低音量氛围 + 真实拟音；爆点机制型用强起拍、突然静音、重低音落点或短促音效做转折。BGM 全片保持同一音乐世界观，只做层次递进或节拍变化，不频繁换曲风。
- 旁白策略先跟传播主线：功能卖点、促销转化、使用场景、信息流广告用信息型旁白解释利益点，默认 3-5 句主体旁白 + 1 句结尾 slogan；氛围种草、品牌理念、高端形象、香氛、珠宝、礼赠用情绪型/诗性旁白承接画面，默认 2-3 句诗性主体旁白 + 1 句结尾 slogan。诗性旁白和 slogan 是固定搭配，不是可选关系；诗性旁白必须形成清晰但低密度的音频线索，只在 hook、情绪转折、packshot 前后等关键落点出现，不逐镜头解释画面，不要求覆盖多数镜头。ASMR、极简、高端或快剪都不能由模型自行取消旁白。
- 旁白音色必须匹配主题调性和品类：高端大气用低声线、气息稳定、语速偏慢；科技/3C 用清晰沉稳、理性可信、参数少而准；医研护肤用专业清晰、克制不过度销售；真实生活/小红书感用亲切自然、像朋友推荐；快剪/促销/信息流用明亮有冲击、短句、结果前置；质地感官用耳语感或轻声短句；森林疗愈/国风雅致用低声舒缓、留停顿。只有用户明确要求无旁白、纯音乐或纯鼓点时才不用人声；必须先定义统一 `voice_profile`，不得每段切换不同 TTS 感声线。
- 人声不要全程铺满，但不能只在结尾出现一句 slogan；高端片少词、低声线、留停顿，快剪片短句、强节奏、结果前置，证明片清晰可信，感官片优先耳语感短旁白。人声优先级高于 BGM，BGM 不盖旁白；旁白、人物台词或口播原文进入 prompt 时必须用 `{具体台词}` 包裹；旁白还要绑定 `voiceover_scene_map`，不强制画面生成字幕。
- 氛围空镜允许短旁白，但每个空镜最多 1 句，优先 6-14 个中文字符或一个短分句，语义必须服务主线，不得用旁白补救无产品画面；结尾 packshot 必须用一句品牌/产品 slogan 口播收束，slogan 只能作为补充收束，不能替代主体旁白。
- 拟音只绑定关键动作，并在 prompt 中用 `< >` 包裹：开盖、倒入、切开、涂抹、按键、布料摩擦、产品落桌、包装撕开；拟音只增强真实感，不抢 BGM 和人声。
- 声音落点服务画面：产品 hero、证明动作、结果状态、最终 packshot 都要有可感知的音乐/拟音节拍点。
- 音频混音必须统一：禁止音量忽大忽小、旁白忽远忽近、BGM 风格断裂、拟音过响、机械 TTS、旁白与画面节奏错位；全片保持统一响度、空间感、混响和节奏落点。

## 卖点编排

单条 15 秒只聚焦 1 个核心卖点 + 2 个辅助卖点。不要堆参数，不编造不可证明功效。

- Hook：痛点、反差、场景欲望、质感冲击、结果前置任选一种。
- 证明：用看得见的动作证明卖点，如倒入、切开、涂抹、启动、上身、收纳前后。
- 利益翻译：把功能翻成用户收益，如省时间、显质感、更顺手、更好搭、更有食欲。
- 结尾：产品 packshot + 品牌/产品记忆点；CTA 只做软性表达，不喊“快买”。

### 功能型产品 3+ 卖点拆解

科技、3C、剃须刀、厨具、小家电、工具、清洁、运动装备等功能型产品，不能只拍“产品很酷”。必须先拆出至少 3 个可见卖点，并围绕卖点设计镜头。

拆解结构：

```text
核心场景痛点 -> 卖点1 功能/结构 -> 卖点2 效率/体验 -> 卖点3 材质/安全/便携/清洁 -> 使用结果 -> packshot
```

每个卖点必须包含：

- `selling_point_name`：卖点名称，如贴面顺滑、快速充电、防水清洗、低噪运行、强劲动力。
- `visible_proof_action`：可见证明动作，如剃须刀贴合下颌线滑过、水流冲洗刀头、磁吸充电座吸附、厨具切开食材、吸尘器吸走碎屑。
- `benefit_translation`：用户收益，如更省时间、少拉扯、更干净、更安心、更好收纳。
- `shot_mapping`：对应镜头编号，至少 1 个镜头服务 1 个卖点；重要卖点可以用 2 个镜头连续证明。

功能型产品默认 8-10 镜头排布，复杂功能或信息流广告可扩展到 10-12 镜头：

| 镜头 | 目标 | 卖点关系 |
|---|---|---|
| 01 痛点/结果 hook | 展示使用前痛点或使用后结果 | 引出核心需求 |
| 02 场景建立 | 展示桌面、厨房、浴室、户外等使用环境 | 明确使用语境 |
| 03 产品 hero | 清晰展示产品外观、结构和关键部件 | 建立产品主体 |
| 04 卖点1 结构/性能证明 | 微距或结构细节，展示核心硬件 | 证明核心能力 |
| 05 卖点1 动作验证 | 让产品完成一次可见动作 | 证明不是只拍外观 |
| 06 卖点2 使用体验证明 | 手部/人物真实使用 | 证明效率、舒适、顺手 |
| 07 卖点3 场景/维护/安全证明 | 防水、清洁、收纳、续航、便携、低噪等 | 证明长期使用价值 |
| 08 结果/对比反馈 | 使用后状态、前后对比或人物反馈 | 翻译成用户收益 |
| 09 规格/配件/组合信息 | 展示容量、档位、配件、系列或套装 | 补齐购买信息 |
| 10 产品稳定 packshot | 产品稳定收尾，logo/结构清晰 | 留下产品记忆 |

剃须刀示例卖点池：

- 贴面顺滑：刀头沿下颌线移动，皮肤没有拉扯表情。
- 多向浮动刀头：微距展示刀头随脸部曲线轻微摆动。
- 防水清洗：水流冲洗刀头，水珠顺着金属/塑料表面滑落。
- 快速充电/续航：充电座吸附、指示灯亮起，但不生成乱码屏幕。
- 低噪舒适：用音效和人物放松表情表达，不生成夸张分贝数据。
- 便携收纳：旅行包、浴室台面、磁吸保护盖或收纳盒。

科技/3C 示例卖点池：

- 速度/响应：屏幕或设备状态变化，但避免假 UI 乱码。
- 续航/充电：充电动作、灯光反馈、桌搭场景。
- 便携/轻量：单手拿起、通勤包取出、桌面切换。
- 连接/兼容：设备靠近、配对反馈、工作流串联。
- 材质/工艺：金属边框、玻璃、按键、接口微距。

厨具/小家电示例卖点池：

- 效率：切/搅/煎/蒸动作缩短流程。
- 易清洁：冲洗、擦拭、拆卸、收纳。
- 安全/稳定：握持、防滑、自动停止、隔热。
- 结果质量：食物切面、熟度、蒸汽、整洁台面。

功能型产品禁区：

- 只拍外观、不证明功能。
- 只说参数、不翻译成用户收益。
- 用不可验证数字、医疗/安全承诺或夸张极限场景。
- 屏幕、指示灯、标签出现乱码。
- 功能动作不完整，如只出现结果、不出现操作过程。

## 分镜与 Prompt 交付

先设计给用户看的分镜表，再编译视频 prompt；用户侧保留时间段，Seedance 侧用镜头顺序表达，避免把精确秒数强塞进镜头文本。

给用户的分镜表至少包含：

```text
时间 | 每秒画面/动作 | 产品卖点 | 主体动作 | 产品保留点 | 镜头轨迹 | BGM | 人声/旁白 | 拟音
```

每个镜头只保留必要执行字段：

- `shot_number/time_range/shot_goal`：编号、时间段和镜头目标。
- `visual_action`：主体、动作、景别、构图、光影和可见卖点证明。
- `product_lock`：产品位置、朝向、logo/SKU/材质/比例保留点。
- `motion_audio`：运镜、速度策略、转场、BGM、拟音、人声/旁白落点和 `voiceover_scene_map`。
- `risk_notes`：无字幕、无水印、无假 logo、无包装漂移、无不合理动作。

Seedance 最终 prompt 必须采用导演稿六段式，不把字段表或 YAML 原样丢给视频模型：

```text
Style & Mood:
说明广告质感、平台画幅、光线、机位组合、剪辑节奏和整体情绪；如涉及参考视频改写，写清哪些机位/节奏来自参考，哪些已按产品重建。

Narrative Summary:
用一句话概括 15 秒 FULL_ARC：开场如何停滑，产品如何出现，卖点如何被证明，结果/情绪如何落地，最终如何 packshot 收束。

Dynamic Description:
按 Cut 1/Cut 2/Cut 3... 写镜头顺序。每个 Cut 写清景别/机位、产品数量与朝向、主动作、手部接触点、微动作、速度策略、转场和音频触发。常规路线不得少于 8 个 Cut；强信息/快剪/科技/功能/信息流路线优先写 10-12 个 Cut；极简留白路线也至少 7 个 Cut。

Static Description:
写固定场景、背景材质、光源方向、道具关系、产品陈列规则和空间连续性。全片默认同一光线逻辑，不因切镜漂移到无关空间。

Audio:
沿用既有声音规则：BGM 用()，拟音用<>，旁白/人物口播用{具体台词}，默认不生成字幕。信息型/功能型/促销型路线默认 3-5 句主体旁白 + 1 句结尾 slogan；诗性/氛围/高端/香氛/珠宝/礼赠路线默认 2-3 句诗性主体旁白 + 1 句结尾 slogan。写清 voiceover_scene_map：每句台词 -> Hook/产品出现/卖点证明/使用结果/packshot，并写清人声能量、语速、情绪、BGM 速度和拟音落点。

Constraints:
收束产品一致性、无字幕、无水印、无乱码、无假 logo、不复制参考视频品牌资产；如用户提供参考广告视频，只抽象其结构、节奏、镜头关系、光影和声音策略，不得复制具体品牌、字幕排版、画面文字或专属视觉资产。
```

编译时只保留必要决策：

- 开头写 `product_lock/reference_asset_roles/reference_usage_decision/platform_ratio_decision/communication_mainline/primary_message/secondary_message/avoid_message_drift/tone_route/category_route/structure_template/lens_count_strategy/audio_strategy/selling_points_breakdown`。
- 参考视频改写时补充 `reference_structure/reference_pov/reference_motion/reference_emotion_arc/reference_audio_logic/reference_risk/rewrite_mode`，只迁移结构、机位、节奏、情绪和声音关系。
- 主传播目标只选 1 个，辅助目标只选 1 个；`primary_message` 必须有 2 个以上镜头证据，无法被画面证明就改写。
- 先定调性，再定证明动作、镜头数量、氛围比例、产品露出节奏和声音设计；禁止只按品类套模板。
- 有产品图时先定义主体：`将图片1中的产品定义为产品主体`；后续涉及产品都写同一主体标签，并保持包装/logo/SKU/材质/比例一致。
- 护肤/医研先做品类纠偏，不因 black tea、peach、mango、rose 等原料名转成食品饮品或香氛；功效只能写用户提供或可见肤感，不编造医疗承诺、百分比数据或夸张前后对比。
- 功能型产品先拆 3+ 可见卖点，每个卖点写 `selling_point_name/visible_proof_action/benefit_translation/shot_mapping`，镜头围绕证明动作设计。
- 氛围型路线允许 1-2 个同类型元素空镜，但空镜后必须回到产品、使用动作或卖点证明，不能连续空镜超过 2 个。
- 快剪路线优先 10-12 个 Cut，每 2-3 个快切后给一次产品清晰锚点；最后 1 秒必须稳定 hold：产品正面、logo/包装清楚、光线稳定、无新动作突变。
- 每条 15 秒至少使用 2 种真实运镜策略，并至少包含 1 次速度变化设计；速度变化必须服务产品信息、卖点证明、情绪停顿或声音落点。
- 若需要广告大字，只能是 2-8 个中文字符并用中文双引号包裹，例如 `“真香”`；大字不是字幕，不逐句复述旁白，并写清内容、颜色、字体风格、出现方式、时机和位置。

prompt 末尾第一句必须写：

```text
保持无字幕，避免生成任何文字或字幕；no subtitles, no captions, no sticker text, no watermark; if intentional big ad words appear, every phrase must be short Chinese text wrapped in Chinese double quotation marks like “...”, otherwise no on-screen text
```

## 最终质量约束

每个 prompt 结尾加入一段统一质量约束，避免重复堆字段：

```text
广告级产品营销短片，真实摄影质感，产品是绝对主角，镜头语言清楚，光影与场景一致，产品包装稳定不变形，logo/标签清晰不漂移，产品比例跨镜头一致，反光和阴影符合真实光源，手部接触点自然，有重量感和生活动作闭环。画面清晰、曝光准确、白平衡准确，材质细节和关键卖点动作可见，BGM 与人声情绪匹配但不盖过产品信息。
不要 CG 塑料感、不要产品悬浮、不要包装变形、不要假 logo、不要乱码标签、不要字幕、不要 caption、不要贴纸字、不要水印、不要低清、不要闪烁、不要跳帧、不要只靠推拉变焦、不要同一镜头堆叠过多运镜、不要人物抢产品主角、不要危险或不合理使用动作、不要手指畸形、不要产品尺寸忽大忽小。
```

## 工具调用

只有主 MD 完成产品上传门控和本轮确认后，才可以进入本节。确认前不得调用视频工具，不得输出 `视频已生成`。

- `prompt` = 六段式完整时间轴 prompt。
- `ref_images` = 用户上传产品/人物/场景/风格图，或主 MD 路由来的 ref 中提取出的图片；产品图始终最高优先级，必须结合 `reference_usage_decision` 判断是否引用原图场景。
- `ref_text/ref_assets` = 文本、视频、音频和图片素材先完成职责路由，再进入参考资产、传播主线、创意视觉解法、音频策略、产品锁定或卖点拆解。
- `duration` 默认 `15`；`ratio` 先按投放平台和用户 prompt 推断：抖音/小红书/快手/视频号/竖屏信息流默认 `9:16`，电视/TVC/Web 端/官网/YouTube/横版展播默认 `16:9`，电商详情/方形卡片可用 `1:1`，无法判断平台时再默认 `9:16`。
- 有用户产品图或首帧图且要强一致时优先 `image_to_video`；无可靠图片参考、概念片或只需文生视频时用 `text_to_video`；本版本不主动生成分镜图。
- 返回字段优先读取 `video_url`、`output_url`、`url`、`result_url` 或本地视频路径；没有可用链接则按失败处理，不伪造链接。

| # | 时间 | 每秒画面/动作 | 产品卖点 | 主体动作 | 产品保留点 | 镜头轨迹 | BGM | 人声/旁白 | 拟音 |
|---|------|---------------|----------|----------|------------|----------|-----------|------|------|
| 1 | 0-2s | ... | ... | ... | ... | ... | ... | ... | ... |

```

```

## 内部自检

- **门控**：是否已完成主 MD 本轮确认；确认前是否没有调用视频工具、没有输出生成成功。
- **参考资产**：是否写清 ref 职责和使用决策；是否只迁移可用结构、机位、节奏、光影和声音关系，剥离原品牌、字幕、水印、平台 UI、人物身份和专属资产。
- **产品与品类**：产品图是否最高优先级；包装/logo/SKU/材质/比例是否跨镜头一致；护肤/医研是否完成品类纠偏；功能型产品是否拆 3+ 可见卖点。
- **镜头结构**：镜头数量是否符合 8-10、10-12 或至少 7 的规则；是否覆盖 hook、产品 hero、细节证明、卖点动作、使用/结果、规格/SKU 或组合信息、最终 packshot。
- **连续性**：TVC 是否保持相对一致的主场景和视觉世界观；多场景时是否有统一调性、产品锚点和转场因果。
- **运动与节奏**：是否至少 2 种运镜策略、1 次速度变化；快剪是否每 2-3 个切点回到产品清晰锚点；最后 1 秒是否稳定 hold。
- **字幕与音频**：是否默认无字幕、无 caption、无贴纸字、水印和乱码；广告大字是否短、稳、用中文双引号；旁白是否不只剩结尾 slogan，旁白原文是否用 `{具体台词}` 并绑定 `voiceover_scene_map`，音色是否匹配品类和调性，BGM/人声/拟音是否统一。
- **Prompt 与工具**：是否使用六段式导演稿；是否没有把字段表原样丢给视频模型；工具是否返回真实可用视频链接或路径，并已调用 `notify_hunman` 展示。
